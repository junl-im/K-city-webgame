# Alpha 0.44 Opening Illustration + Silent In-App Pass

## 목표

0.44는 카카오톡/인앱 브라우저 대응 로직을 유지하되 사용자 화면에는 개발자용 안내 문구가 보이지 않도록 정리하고, 첫 시작 화면에 오프닝 캐릭터 일러스트를 추가한 패치입니다.

## 변경 사항

- 카카오톡/인앱 브라우저 감지 및 가로 뷰포트 안전 클래스 유지
- 브라우저 대응 안내 토스트/하단 배너 제거
- 첫 시작 화면 문구에서 브라우저 정책 안내 제거
- 신규 오프닝 캐릭터 일러스트 `title-hero-legend.svg` 추가
- 시작 화면 배경, 영혼 링, 룬, 버튼 광택 애니메이션 정리
- 가로 뷰포트에서도 시작 화면이 과도하게 잘리지 않도록 별도 레이아웃 추가
