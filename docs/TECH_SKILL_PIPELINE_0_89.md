# Soul Online Alpha 0.89 - Skill Pipeline and Technical Polish

0.89는 스킬 화면을 다음 단계 구조로 옮기기 위한 안정화 패치다. 단순히 버튼을 추가하는 대신, 스킬 성장 상태, 강화 확인 UX, 에셋 예열 우선순위, safe-frame 점검 범위를 함께 다룬다.

## 변경 목표

- 스킬 성장 UI를 `skillPanelRenderer089.ts`로 일부 분리한다.
- 스킬 숙련 강화는 즉시 소비가 아니라 확인 패널을 거치게 한다.
- 사용자가 강화 전 비용, 다음 효과, 조건 부족 이유를 확인할 수 있게 한다.
- 에셋 예열은 고정 URL만 불러오지 않고 현재 직업/화면 상태에 따라 우선순위를 만든다.
- System Doctor에 스킬 성장 상태와 에셋 예열 상태를 표시한다.

## 기술 점검

- UI 화면 이탈 검사에 스킬 강화 확인 패널과 스킬 연결성 스트립을 추가했다.
- 저성능 상태에서는 0.89 장식 애니메이션도 `perf-reduced-motion-089`로 줄인다.
- 스킬 강화 버튼은 `data-town-upgrade-skill`에서 바로 소비하지 않고 `data-skill-confirm-089` 확인 단계로 연결된다.
- 0.88 상점 확인 UX와 0.89 스킬 확인 UX가 같은 방향의 실수 방지 패턴을 공유한다.

## 다음 단계

0.90에서는 카드/소울 화면 렌더러 분리, 카드 세트 장착 UX, 대형 스프라이트 시트 로딩 우선순위 개선을 이어가는 것이 좋다.
