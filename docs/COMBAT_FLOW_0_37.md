# 0.37 Combat Flow Pass

## 목적

0.36에서 정리한 UI 안정성을 유지하면서, 사냥 반복 플레이의 손맛과 보상 피드백을 강화한다.

## 주요 변경

### 1. 전투 연쇄 시스템

- 몬스터 처치 시 체인 타이머가 시작된다.
- 제한 시간 안에 다음 몬스터를 처치하면 체인 수가 증가한다.
- 체인 수에 따라 EXP/GOLD 보너스가 증가한다.
- 보너스는 과도한 인플레이션을 막기 위해 상한을 둔다.

### 2. 정예 몬스터

- 일반 몬스터가 확률적으로 정예 개체로 변한다.
- 정예 개체는 다음 접두어 중 하나를 가진다.
  - 분노한
  - 고대의
  - 신속한
  - 저주받은
- 정예는 체력/공격/방어/보상 계수가 증가한다.
- 필드 렌더링에서는 오라와 이름색으로 구분한다.
- 타겟 카드도 정예 상태일 때 강조된다.

### 3. UI 영향 최소화

- 새 HUD는 우측 작은 `field-chain-meter`로 추가했다.
- 마을 UI에는 영향을 주지 않는다.
- 미니맵 비활성화 정책은 유지한다.

## 검증

- `npm run build` 통과 필요.
- 필드 입장 후 몬스터 처치 시 체인 표시가 활성화되어야 한다.
- 정예 몬스터 타겟 시 타겟 카드가 강조되어야 한다.
