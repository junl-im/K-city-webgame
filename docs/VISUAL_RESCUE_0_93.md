# Soul Online Alpha 0.93 Visual Rescue

0.93은 진단 기능을 더 얹기보다, 실제 사용자가 보는 화면의 배치와 색 대비를 먼저 복구하는 패치입니다.

## 수정 목표

- 첫 시작 화면을 원화 배경 + 중앙 TOUCH TO START 흐름으로 복구
- 누적된 구형 마을 UI 레이어가 동시에 보이는 문제 차단
- 마을 화면을 하나의 `town-master-lobby-074` 기준 모바일 safe-frame 안에 재배치
- 사냥터 HUD, 퀘스트 트래커, 미니맵, 조이스틱, 스킬/공격 버튼이 화면 밖으로 나가지 않도록 고정
- 닫기 버튼에 크리스탈 계열 이미지 에셋 적용
- 밝은 패널/어두운 필드에 맞춰 글자 대비 보정

## 핵심 파일

- `src/styles/alpha093.css`
- `src/ui/visualRescue093.ts`
- `src/main.ts`

## 주의

이 패치는 CSS 중복을 삭제하는 패치가 아니라, 누적된 과거 UI 레이어를 강하게 누르고 한 화면만 정상 노출시키는 복구 패치입니다. 다음 단계는 실제로 오래된 CSS/HTML 레이어를 제거하는 정리 작업입니다.
