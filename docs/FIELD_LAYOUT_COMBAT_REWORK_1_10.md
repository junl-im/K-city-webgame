# Soul Online Alpha 1.10 - Field Layout & Combat Rework

## 목적

1. 사냥터 UI 겹침을 우선 제거한다.
2. 미니맵은 모바일 세로 화면에서 과밀을 만들기 때문에 1.10부터 숨긴다.
3. 몬스터 타겟 카드는 플레이어 HUD 바로 아래 중앙으로 고정한다.
4. 골드/소울/전투력 리소스 스트립은 필드 화면에서 제거한다.
5. 프로필 이미지는 직업/성별별 실제 초상화로 채운다.
6. 우측 하단 기능키/메뉴키/공격키/스킬키를 서로 겹치지 않는 그리드로 재배치한다.
7. 몬스터 난이도와 개체 수를 높이고, 자동사냥이 여러 포켓을 이동하게 만든다.
8. 초중반 사냥터를 추가해 사냥 루프 다양성을 늘린다.

## 주요 변경 파일

- `src/styles/alpha110.css`

## UI 정리

- 첫 시작 화면은 원화 배경 + 중앙 `TOUCH TO START` 버튼 기준으로 재정렬했다.
- 마을 패널은 난잡한 색상을 줄이고 파치먼트/청색/골드 기준으로 정리했다.
- 사냥터 미니맵과 리소스 스트립은 필드 화면에서 숨겼다.
- 몬스터 타겟은 HUD 바로 아래 중앙 고정으로 옮겼다.
- 퀘스트 카드는 좌측 상단으로 분리했다.
- 조이스틱은 좌측 하단, 물약은 중앙 하단, 스킬은 우측 세로 스택, 공격/기능키는 우측 하단 그리드로 분리했다.

## 전투/사냥 개편

- 몬스터 밀도를 높였다.
- 몬스터 스탯은 사냥터 권장 레벨과 플레이어 레벨 차이에 따라 더 강하게 스케일링한다.
- 자동사냥은 단순 최단 거리 타겟만 반복하지 않고, 사냥터 내 포켓 경로를 순환하도록 보정했다.
- 처치 후 다음 포켓으로 이동하도록 자동 루트 인덱스를 갱신한다.
- 초중반 사냥터 5개를 추가했다.

## 추가 사냥터

- 해바라기 초원
- 안개 낀 강가
- 무너진 감시마당
- 메아리 등불숲
- 무너진 수도교

## 남은 과제

- 실제 모바일 캡처 기반 미세 좌표 조정
- CSS 본체 다이어트
- 필드 HUD를 DOM 대신 Pixi 오버레이 일부로 이전하는 장기 최적화
- 보스 패턴과 몬스터 배치의 난이도 곡선 추가 튜닝
