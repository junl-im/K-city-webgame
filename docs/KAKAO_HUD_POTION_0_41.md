# Soul Online Alpha 0.41 - Kakao Browser HUD + Potion Pass

## 목표

0.40 기준으로 카카오톡 인앱 브라우저, 필드 HUD 배치, 접속 플로우, 캐릭터 프로필 이미지, 물약 시스템을 정리한 패치입니다.

## 적용 내용

- 카카오톡 인앱 브라우저 감지 로직 추가
- 전체화면/화면 방향 잠금 실패 시에도 플레이 가능한 안전 HUD 배치 적용
- forced landscape 상태에서 orientation guard가 화면을 막지 않도록 처리
- 캐릭터 선택 후 `마을 입장` 단계를 거치지 않고 즉시 마을 접속
- 신규 캐릭터 생성 후에도 바로 마을 접속
- 필드 캐릭터 정보창을 상단 전체 폭으로 확장
- 몬스터 타겟창을 작게 줄이고 캐릭터 정보창 아래 중앙 배치
- 퀘스트 진행창을 우측 상단으로 이동
- 우측 상단 메뉴 버튼을 우측 하단 액션 버튼 묶음으로 이동
- 우측 하단 액션 배열 정리
  - 하단 1열: 자동 / 공격 / 메뉴
  - 메뉴 위: 가방
  - 가방 위: 절전
  - 액션 버튼 위쪽에 스킬 버튼 3개 배치
- 필드 HP/MP 물약 단축 버튼 추가
- 하급/중급 HP/MP 물약 아이템 추가
- 자동사냥 중 HP/MP가 낮으면 하급 물약 자동 사용
- 마을 상점에 물약 판매 추가
- 인벤토리/상세창에서 물약 사용 가능
- 캐릭터 HUD, 마을 상단 카드, 계정 프로필에 실제 직업/성별 초상화 표시

## 기술 메모

- Fullscreen API와 Screen Orientation API는 모바일 브라우저 정책에 따라 사용자 입력, fullscreen/PWA 컨텍스트 등이 필요할 수 있으므로, 카카오톡 인앱 브라우저에서는 실패를 정상 케이스로 보고 CSS safe mode로 우회합니다.
- 저장 구조는 물약 아이템과 스냅샷 물약 카운트가 추가되어 `SAVE_VERSION = 29`로 올렸습니다.
