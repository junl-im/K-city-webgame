# 0.40 Gear Resonance Pass

0.40은 0.39 Skill Mastery Pass 이후 장비 파밍 목표를 강화한 개발 패치입니다.

## 핵심 변경

- 장비 공명 시스템 추가
  - 무기, 방어구, 유물을 모두 장착하면 기본 공명 발동
  - 세 부위의 최저 등급에 따라 R / SR / SSR / UR 등급 공명 발동
  - 세 부위의 최저 강화 단계에 따라 +5 / +10 / +15 / +20 강화 공명 발동
- 장비 공명 보너스가 실제 전투 능력치에 반영됨
- 마을/필드 인벤토리에 장비 공명 요약 패널 추가
- 캐릭터 정보창에 활성 공명 요약 표시
- 마을 인벤토리에 중복 장비 분해 버튼 추가
  - 장비별 최소 1개는 보존
  - 초과 수량만 골드, 소울 파편, 강화석으로 전환
- 저장 구조 변경 없음
  - SAVE_VERSION은 28 유지

## 의도

0.39에서 스킬 숙련으로 성장 루프가 추가되었으므로, 0.40은 장비 파밍이 단순 획득/강화에서 끝나지 않도록 세 부위 세팅 목표를 제공한다.
