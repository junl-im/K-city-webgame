# Soul Online alpha 0.98 - 대형 UI 클린업

## 목표

0.70~0.97 동안 누적된 타이틀/로그인/마을/필드 UI 덮어쓰기 레이어를 계속 추가하지 않고, 실제 플레이 화면에서 하나의 클린 레이어만 보이도록 정리했다.

## 적용 내용

- 타이틀 화면을 원화 배경 + 단일 TOUCH TO START 버튼 구조로 강제 정리했다.
- 로그인/서버/캐릭터 흐름은 모바일 카드형 레이아웃으로 다시 정렬했다.
- 마을은 신규 `town-clean-hub-098` 셸을 주 화면으로 사용하고, 구형 0.70/0.72/0.74/일반 town layout은 화면에서 제외한다.
- 기존 데이터 바인딩은 유지하기 위해 `data-town-lobby-*`, `data-town-content`, `data-zone-id` 속성을 새 허브에 연결했다.
- 사냥터 HUD, 퀘스트, 미니맵, 조이스틱, 물약, 스킬, 공격 버튼을 모바일 safe-area 안쪽으로 다시 고정했다.
- 닫기 버튼은 0.98 크리스탈 에셋으로 통일했다.
- `alpha087~alpha097.css` import를 제거하고 `alpha098.css` 중심으로 클린 레이어를 다시 잡았다.

## 남은 과제

- `styles.css` 자체가 여전히 크므로 다음 단계에서는 실제 CSS 삭제/분리가 필요하다.
- `main.ts`에는 마을/시트/상점/스킬 로직이 아직 많이 남아 있으므로 렌더러 분리를 계속해야 한다.
- 모바일 실제 기기에서 360×740, 390×844, 430×932, 태블릿 폭을 QA해야 한다.
