# Soul Online alpha 1.06 - 성능/스프라이트 아틀라스 대규모 최적화

## 목표
- 모바일 세로모드 플레이에서 사냥터 렉을 줄인다.
- 2MB 이상 대형 캐릭터/몬스터 WebP Atlas 로딩 부담을 낮춘다.
- HUD, 몬스터 정보, 퀘스트, 스킬/공격 버튼이 화면 밖으로 나가지 않게 한다.
- 밝은 UI와 어두운 전투 HUD의 글자 대비를 분리한다.

## 주요 변경
- `public/assets/soulpack-lite/`에 50% 저용량 캐릭터/몬스터 스프라이트 Atlas 추가.
- 저메모리, 저코어, 데이터 절약, 좁은 세로 화면에서는 자동으로 Lite Atlas를 우선 로딩.
- `SpriteSheetAnimator`가 실제 Atlas 해상도를 감지해 50% Atlas도 원래 크기로 렌더링한다.
- 텍스처 로딩 우선순위를 플레이어/현재 사냥터 몬스터 중심으로 정렬.
- PixiJS 렌더 예산을 1.06 기준으로 더 보수적으로 조정.
- `alpha106.css`에서 사냥터 HUD와 조작 버튼 레이아웃을 다시 고정.
- System Doctor에 `1.06 스프라이트/성능` 항목 추가.

## 남은 과제
- `styles.css` 본체가 여전히 크므로 실제 미사용 CSS 삭제가 필요하다.
- 캐릭터/몬스터 Atlas를 장기적으로는 액션 단위/방향 단위로 분할하는 것이 가장 좋다.
- 실제 기기에서 360/390/430px 세로 화면 QA가 필요하다.
