# 0.34 UI Stabilization Pass

0.34는 0.32 그래픽 UI 스킨과 0.33 필드 패치 이후 무너진 화면 배치를 먼저 안정화하는 패치입니다.

## 확인한 주요 문제

- `index.html`의 마을 사냥터 패널 끝부분에 불필요한 닫힘 태그가 있어 이후 패널 배치가 브라우저 보정에 의존했습니다.
- `styles.css` 하단에 여러 버전의 `!important` 규칙이 누적되어 같은 요소가 서로 다른 위치/크기로 반복 지정되었습니다.
- 0.32 그래픽 스킨이 버튼, 패널, 슬롯, HUD 전체에 광범위하게 적용되면서 작은 HUD 막대와 슬롯 텍스트까지 과하게 커졌습니다.
- 캐릭터 HP/MP HUD, 몬스터 타겟 카드, 퀘스트 트래커가 모바일 너비에서 서로 침범할 가능성이 있었습니다.
- 미니맵은 제거 정책 이후에도 과거 규칙이 남아 다시 살아날 수 있는 상태였습니다.
- 인벤토리/카드/스킬 슬롯은 너무 큰 카드형과 너무 작은 정사각형 규칙이 동시에 남아 이미지 표현이 불안정했습니다.

## 수정 내용

- 마을 사냥터 패널의 잘못된 HTML 닫힘 구조 수정
- 필드 HUD 레이어 재정렬
  - 캐릭터 HP/MP/EXP: 좌측 상단
  - 재화 스트립: 캐릭터 HUD 아래
  - 몬스터 타겟: 우측 상단
  - 진행 퀘스트: 타겟 아래
- 720px 이하 모바일 전용 배치 추가
  - 캐릭터 HUD는 상단 전체 폭
  - 재화는 한 줄
  - 퀘스트/타겟은 좌우 2분할
- 380px 이하 초소형 화면에서는 타겟과 퀘스트를 세로 배치
- 미니맵 완전 비활성화 고정
- 스킬/액션 도크 크기와 위치 재조정
- 시트와 마을 드로어의 `max-height`, 내부 스크롤, z-index 정리
- 카드/인벤토리/스킬 슬롯 크기 재조정
  - 이미지 영역을 유지하되 슬롯 전체가 과하게 커지지 않도록 조정
  - 버튼과 희귀도 라벨이 항상 보이도록 고정
- PWA 캐시와 앱 버전을 0.34로 갱신
