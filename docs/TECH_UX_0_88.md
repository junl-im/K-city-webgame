# Soul Online Alpha 0.88 - Technical UX Stabilization

## 목표

0.88은 가방/상점처럼 유저가 반복적으로 누르는 화면의 연결성과 조작 실수를 줄이는 패치다. 기능을 크게 늘리기보다 구매 확인, 필터 상태, 렌더 예산 점검을 추가해 모바일 웹게임의 안정성을 높인다.

## 변경점

- 가방 필터: 전체, 장비, 소모품, 재료, 스킬서.
- 상점 필터: 전체, 소모품, 스킬북, 장비, 재료.
- 상점 구매 확인 패널 추가.
- 상점 재고 표시 목록과 실제 가격 테이블을 `TOWN_SHOP_STOCK_088` 기준으로 통합.
- `lumina-seal`, `radiant-ore` 구매 연결 누락 수정.
- `townPanelRenderer088.ts`로 반복 UI 렌더링 일부 분리.
- `alpha088.css`로 신규 UI 스타일 분리.
- System Doctor에 DOM node, 이미지 수, 표시 패널 수 기반 렌더 예산 점검 추가.

## 기술 체크

- 빌드 통과가 1차 안전 기준이다.
- UI 이탈 검사는 기존 0.87 safe-frame에 0.88 신규 필터/구매 패널을 추가한다.
- 필터 상태는 메모리 상태로 유지한다. 저장 데이터에는 영향을 주지 않는다.
- 구매 확정 전까지 골드/가방/스킬 데이터는 변경하지 않는다.

## 다음 권장 작업

0.89에서는 스킬 렌더러 분리, 스킬 강화 확인 UX, 에셋 프리로드 우선순위 분리가 필요하다.
