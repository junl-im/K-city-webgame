# Alpha 0.55 Quality Polish Pass

0.55는 0.54에서 늘어난 아이템/드랍/필드/모션 요소를 한 번 더 정리하는 품질 패스입니다.

## 핵심 목표

- 슬롯은 그림이 먼저 보이고, 글자는 보조 정보만 남도록 정리
- 가방은 장비/소모품/재료/스킬서를 희귀도와 장착 상태 기준으로 정렬
- 타격 시 이펙트가 더 자주 터져 액션감을 보강
- 필드 바닥은 소품 세트 드레싱과 빛길로 덜 허전하게 보정
- 신규 드랍 재료와 장비를 추가해 장기 사냥 보상 밀도 보강

## QA 메모

- 모바일 390px 이하에서는 가방을 2열로 전환해 슬롯 그림 크기를 확보합니다.
- 고레벨 전선은 수정 화로/균열 제단/빛길을 추가로 배치합니다.
- 혈맹 전선은 표지판/랜턴/제단 계열 소품을 더 많이 사용합니다.
