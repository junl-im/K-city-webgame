# Alpha 0.52 Sovereign Design Overhaul

## 목표

0.52는 리니지식 장기 파밍 구조와 대규모 그래픽 개선을 동시에 진행한 패치입니다. 라우풀/카오틱 성향이 실제 전투 성능과 드랍에 영향을 주도록 만들고, 보스 토벌 보상을 상자 루프로 확장했습니다.

## 주요 변경

- 라우풀 성향 보너스와 카오틱 패널티를 능력치에 반영
- 성향에 따른 드랍 배율 적용
- 필드 진입 지점 안전지대 적용
- 보스 보급 상자, 정예 보스 상자, 고대 보스 전리품, 라우풀 보급 궤짝 추가
- 신규 장비/재료 12종 추가
- 보스 메뉴에 성향/보급/안전지대 가이드 추가
- 타이틀 히어로 일러스트 0.52 SVG 추가
- 마을 파노라마 0.52 SVG 추가
- 전리품 팝업과 슬롯 그래픽 톤 강화

## 검증

- `npx tsc --noEmit` 통과
- `npm run build` 통과
