# Alpha 0.64 Visual Repair Pass

0.64는 0.63 대규모 자산 업그레이드 이후 어색해진 첫 시작 화면과 밝은 에셋 위 텍스트 가독성, 일부 터치 레이어 충돌을 정리하는 안정화 패치입니다.

## 적용 내용

- 첫 시작 화면을 검수 만족도가 높았던 0.60 키비주얼 기반으로 복원
- `TOUCH TO START` 버튼을 모바일 안전영역 기준 중앙 하단 라인으로 재배치
- 타이틀 상단 상태 바를 중앙 정렬형 글래스 캡슐로 재정리
- 밝은 하늘/아이보리 UI 위에서 글자가 흐려지지 않도록 잉크 블루 계열 텍스트 토큰 추가
- 로그인, 서버, 캐릭터, 마을, 필드 HUD, 시트, 아이템 상세 카드의 텍스트 대비 보정
- 장식용 배경/파노라마/디오는 `pointer-events: none`으로 고정
- 실제 조작 버튼과 입력창은 `pointer-events: auto` 및 `touch-action: manipulation`으로 터치 우선 처리
- 마을 핫스팟 레이어보다 실제 패널과 하단 메뉴가 위에 오도록 z-index 정리
- PWA 캐시 버전 `soul-online-alpha-v0-64`로 갱신

## QA 체크

- `npm run build` 통과
- 첫 시작 화면에서 BGM 버튼과 TOUCH TO START 버튼 클릭 가능
- 로그인/서버/캐릭터 선택 버튼 터치 가능
- 마을 패널, 사냥터 카드, 하단 메뉴, 상세 시트가 핫스팟에 가려지지 않음
- 밝은 배경 위 보조 문구가 회색으로 묻히지 않음
