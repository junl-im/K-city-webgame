# Skill Mastery 0.39

0.39 패치는 전투/보상 흐름을 건드리지 않고, 기존 스킬 시스템에 장기 성장 축을 추가한다.

## 목적

0.37의 연쇄 사냥과 0.38의 몬스터 패턴 이후, 플레이어가 같은 스킬을 계속 쓰면서도 성장 체감을 얻도록 스킬 숙련 레벨을 도입했다.

## 주요 변경

- `PlayerSave.skillLevels` 추가
- `SAVE_VERSION = 28`
- `SKILL_MAX_LEVEL = 5`
- `skillMasteryCost(currentLevel)` 추가
- 스킬 숙련 효과
  - Lv당 피해/회복 +14%
  - Lv당 쿨타임 -3.5%
  - Lv당 MP 소모 -4.5%
- 마을/필드 스킬창에서 숙련 강화 가능
- 자동사냥 스킬 사용 조건이 실제 숙련 MP 비용을 기준으로 동작

## 밸런스 메모

숙련 강화 재료는 기존 재화 루프를 사용한다.

- 골드
- 소울 파편
- Lv.4 이상 구간부터 강화석

이 구조는 별도 신규 아이템 없이도 기존 사냥/퀘스트/강화 루프와 연결된다.
