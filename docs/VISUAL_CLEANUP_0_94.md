# Soul Online Alpha 0.94 Visual Cleanup

## 목적
0.90~0.93에서 진단/성능 레이어가 늘어나는 동안 실제 화면 품질이 흔들린 문제를 우선 복구한다. 0.94는 기능 추가보다 실제 플레이 흐름의 보이는 화면을 먼저 정상화하는 패치다.

## 수정 범위
- 첫 시작 화면을 하나의 원화 배경과 하나의 `TOUCH TO START` 버튼만 보이는 구조로 강제 정리.
- 타이틀/로그인/마을 상태에서 필드 HUD가 뒤섞여 보이지 않도록 라우트별 표시 규칙 추가.
- 마을 배경에서 UI 스크린샷 레퍼런스를 직접 배경으로 쓰던 구조를 제거하고, 게임용 원화 배경 에셋으로 교체.
- 마을 화면은 `town-master-lobby-074` 한 레이어만 기준으로 재배치.
- 구형 마을 레이어, 구형 하단 메뉴, 구형 핫스팟 레이어가 같이 보이지 않도록 차단.
- 사냥터 HUD, 퀘스트 트래커, 미니맵, 조이스틱, 물약, 스킬, 공격 버튼을 모바일 safe-area 안쪽으로 재배치.
- 닫기 버튼을 public 경로의 크리스탈 이미지로 고정 적용.
- 밝은 UI는 네이비 텍스트, 어두운 사냥터 HUD는 흰색/금색 텍스트로 대비 보정.

## 남은 과제
- 0.95부터는 CSS를 더 추가하는 방식보다 오래된 타이틀/마을 레이어 자체를 삭제해야 한다.
- `styles.css`의 누적 레거시 규칙이 여전히 많기 때문에 실제 CSS 다이어트가 필요하다.
- 사냥터 전투 UI는 0.94에서 화면 이탈을 막는 수준이며, 이후 전투 전용 HUD 아트 리디자인이 필요하다.
