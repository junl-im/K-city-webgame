# Soul Online Alpha 1.25 - 필드 런타임 다듬기 패치

## 목적

1.24에서 복구한 고해상도 2.5D 에셋을 유지하면서, 사냥터 진입과 필드 조작 중 흔들리는 부분을 안정화합니다.

## 핵심 변경

- 사냥터 입장 중 중복 클릭/중복 전환을 차단하는 `fieldRuntimePolish125`를 추가했습니다.
- 같은 2.5D 시트가 빠른 재입장이나 장면 전환 중 다시 요청되지 않도록 `SolGame`에 공유 텍스처 캐시를 추가했습니다.
- 필드 HUD, 조이스틱, 공격/스킬/포션 버튼 충돌 검사를 1.25 레이어에서 다시 throttling해서 매 프레임성 DOM 측정을 줄였습니다.
- START/Login/Town/Field 경로에서 펫, 동행, 룬, 회전 가드, 구형 고스트 레이어가 다시 살아나지 않도록 마지막 CSS 레이어에서 차단했습니다.
- System Doctor에 `1.25 필드 런타임` 항목을 추가했습니다.

## 유지한 방향

- 기존 2.5D 고해상도 에셋 사용을 유지합니다.
- 저화질/고화질 자동 전환은 다시 만들지 않았습니다.
- 첫 실행 경로는 1.20~1.22 복구 커널의 가벼운 구조를 유지합니다.

## 빌드

- PWA 캐시: `soul-online-alpha-v1-25`
- 패키지 버전: `1.25.0`
- `npm run build` 통과
