# Soul Online Alpha 0.28 UI Feedback & 2.5D Polish Pass

## 목표

0.28은 전투/마을 UI의 체감 피드백과 슬롯형 인벤토리의 가독성을 보강하고, 업로드된 지형 레퍼런스를 참고해 필드 오브젝트를 더 2.5D 사냥터 에셋 느낌으로 다듬은 패스입니다.

## 적용 범위

- 구매, 강화, 장착, 주요 버튼 클릭 시 짧은 눌림/확인 피드백 표시
- 스킬서는 한 번 구매/습득하면 상점에서 품절 처리
- 사냥터 EXP 바 최상단 고정
- 타겟 몬스터 정보는 상단 중앙의 작은 카드로 재배치
- 미니맵은 프로필 아래로 내려 프로필 UI와 충돌하지 않게 조정
- 진행중 퀘스트 HUD 클릭 시 완료 보상 또는 다음 퀘스트 사냥터 자동 이동
- 골드/소울젬은 가방 패널 안에서 확인
- 캐릭터 스탯과 환경설정/계정 저장 영역을 분리된 카드로 정리
- 브라우저 기본 confirm 대신 인게임 종료 확인 팝업 사용
- 상세 팝업 z-index를 최상위로 올려 뒤로 깔리는 문제 수정
- 가방/카드/스킬/영혼 슬롯 크기 확대 및 핵심 정보 표시 강화
- 장비/스킬/영혼 아이콘 이미지 추가
- 카드 이미지 재생성
- 업로드한 지형 레퍼런스를 참고한 바위, 나무, 상자, 횃불 오브젝트 재배치

## 다음 후보

- 아이템 상세 비교 UI: 현재 장비와 선택 장비 스탯 비교
- 상점 구매 연출: 아이템이 가방으로 들어가는 이동 이펙트
- 사냥터 오브젝트 충돌/가림 처리 개선
- 몬스터 등장/필드보스 경고 연출
