# Soul Online Alpha 1.16 - 시작 화면/필드 HUD 요청 반영 패치

## 기준
- 기준 전체 ZIP: `SoulOnline-alpha115-full-overwrite.zip`
- 패치 버전: `1.16.0`
- PWA cache: `soul-online-alpha-v1-16`

## 반영 항목

### 1. 첫 로그인 시작 화면
- `TOUCH TO START` 버튼을 화면 중앙권으로 이동했습니다.
- 버튼 아래 중복 보조 문구와 버튼 내부 장식선을 제거했습니다.
- 상단 미니 내비게이션은 스크린샷처럼 둥근 글래스 바 형태로 정리했습니다.
- 타이틀 버전을 `v1.16.0`으로 갱신했습니다.

### 2. 사냥터 필드 밝기 개선
- Pixi 필드에 `addFieldReadabilityWash116()`를 추가했습니다.
- 과하게 어둡던 바닥 그림자 알파를 낮췄습니다.
- DOM 캔버스에도 밝기/채도 보정 필터를 적용해 모바일 화면에서 필드가 더 밝게 보이도록 했습니다.

### 3. 필드 HUD 재배치
- 플레이어 HUD는 좌측 상단에 고정했습니다.
- 퀘스트 미니창은 우측 끝 상단으로 이동했습니다.
- 몬스터 타깃 정보창은 플레이어 HUD 아래로 이동했습니다.
- 마을 복귀 버튼은 HUD 안에서 빼고 조이스틱 위쪽으로 이동했습니다.
- 리소스 스트립은 타깃 카드 아래에 작게 정렬했습니다.

### 4. 하단 조작 UI 충돌 제거
- 조이스틱은 좌측 하단 단독 영역으로 고정했습니다.
- HP/MP 물약 버튼은 중앙 하단 세로 스택으로 정리했습니다.
- 우측 하단 행동 버튼은 2x3 그리드로 재정렬했습니다.
- 스킬 버튼은 행동 버튼 위쪽에 3개 가로 정렬했습니다.
- 초소형 화면에서는 스케일을 낮추고 일부 보조 정보를 숨겨 충돌을 줄입니다.

### 5. 펫 UI 제거
- 아직 정식 펫 시스템이 아니므로 마을의 펫 버튼 DOM을 제거했습니다.
- 타이틀의 임시 companion 장식은 숨김 처리했습니다.
- 필드 플레이어 옆 슬라임 companion 스프라이트 생성을 중단했습니다.

## 추가 진단
- `src/ui/fieldLayout116.ts`를 추가했습니다.
- System Doctor에 `1.16 요청 UI` 항목을 추가했습니다.
- overflow / collision / pet placeholder 노출 상태를 진단합니다.
