# Alpha 0.83 UI System Unification Notes

0.83은 0.81에서 도입한 AI 레퍼런스 기반 판타지 UI 방향과 0.82의 마을 진행 허브를, 실제 사용 빈도가 높은 내부 화면까지 확장하는 패치다.

## 목표

- 가방, 스킬, 상점, 보상 팝업의 시각 언어를 블루/골드 판타지 톤으로 맞춘다.
- 모바일 웹에서 버튼과 카드가 화면 밖으로 밀리는 현상을 줄인다.
- 기존 데이터/세이브/전투 로직은 유지하고 UI 레이어만 안전하게 덧씌운다.

## 적용 내용

### 1. 가방 정비 보드

`town-loadout-board-083`을 추가했다. 장착 슬롯, 전투력, 강화 가능 장비, 핵심 재료를 한눈에 보여준다.

### 2. 스킬 성장 보드

`skill-training-board-083`을 추가했다. 습득 수, 사용 가능 스킬 수, 숙련 합계, 추천 강화 액션을 보여준다.

### 3. 상점 추천 보급

`shop-hero-083`을 추가했다. HP/MP 물약 묶음 구매와 가방 점검 이동을 상점 최상단에서 바로 제공한다.

### 4. 보상 팝업 강화

`loot-pop-083`으로 보상 팝업을 중앙 하단 블루/골드 스타일로 변경했다. SSR/UR 등급 보상은 더 강한 하이라이트를 받는다.

### 5. 모바일 안전영역

가방/스킬/카드 슬롯 그리드는 `auto-fill + clamp()` 기반으로 보정했다. 작은 화면에서는 2열로 고정해 가로 오버플로를 줄인다.
