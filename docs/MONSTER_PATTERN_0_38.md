# Monster Pattern 0.38

0.38 패치는 전투 체감 강화를 위해 고위험 몬스터에게 회피 가능한 경고 패턴을 추가한다.

## 패턴 종류

| 패턴 | 몬스터 | 동작 |
| --- | --- | --- |
| 화염 숨결 | 화염 드레이크, 심연룡 | 몬스터에서 플레이어 방향으로 직선 경고 후 피해 |
| 대지 충격파 | 야전 군주, 흑요석 가디언, 묘지 기사 | 몬스터 주변 원형 경고 후 피해 |
| 그림자 폭발 | 망령 사제, 폭풍 하피 | 플레이어 위치에 원형 경고 후 피해 |

## 설계 의도

- 자동사냥 중에도 피격 경고가 눈에 보이게 한다.
- 수동 조작 시 경고 범위 밖으로 빠지면 `회피` 피드백을 제공한다.
- 보스/정예의 압박은 살리되, 패턴 쿨다운과 선딜을 두어 불합리한 즉사 느낌을 줄인다.

## 안정화 메모

- `WorldMob`에 패턴 쿨다운/선딜/타겟 좌표를 저장한다.
- 패턴 선딜 중에는 기존 추격/일반 공격 AI를 잠시 멈춘다.
- 패턴 처리가 끝나면 추격 또는 복귀 상태로 되돌린다.
