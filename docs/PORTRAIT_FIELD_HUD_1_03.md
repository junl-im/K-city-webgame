# Soul Online Alpha 1.03 - Portrait Lock & Field HUD Rebuild

## 목적

1. 세로모드 전용 게임 흐름을 명확히 고정한다.
2. 사냥터 UI의 핵심 문제를 우선 해결한다.
   - 몬스터 정보 카드가 화면 밖으로 나가는 문제
   - 우측 하단 메뉴/스킬/공격 버튼 겹침
   - 조이스틱/물약/스킬/공격 버튼 간격 부족
   - 배경과 글씨 대비 불일치
   - 모바일 좁은 화면에서 HUD가 과밀해지는 문제
3. 1.02 에셋 키트를 실제 사냥터 HUD에 더 적극적으로 연결한다.

## 세로모드 정책

웹 브라우저는 일반 탭 상태에서 화면 회전을 100% 강제 차단할 수 없다. 그래서 1.03은 3단계로 처리한다.

1. PWA manifest에 `portrait-primary` 적용
2. 터치/시작/전체화면 시 `screen.orientation.lock('portrait-primary')` 요청
3. 브라우저가 가로 회전을 허용하면, 게임 UI를 깨진 채로 보여주지 않고 세로 안내 가드를 표시

## 사냥터 HUD 재배치

- 상단 왼쪽: 플레이어 HUD
- 상단 오른쪽: 미니맵
- 미니맵 아래: 몬스터 정보 카드
- 플레이어 HUD 아래: 재화 바
- 재화 바 아래: 퀘스트 카드
- 왼쪽 하단: 조이스틱
- 중앙 하단: HP/MP 물약
- 오른쪽 하단: 공격 버튼
- 공격 버튼 위쪽: 스킬 버튼 세로 스택
- 공격 주변: 자동/메뉴/가방/절전 보조 버튼

## 성능 보정

- 390px 이하/700px 이하 화면에서는 전투 로그와 콤보 일부를 숨겨 조작 영역을 확보한다.
- 저사양/데이터 절약/3G 환경에서는 `field-lite-103`을 자동 적용한다.
- 그림자/필터/장식 레이어를 줄여 HUD 렌더 비용을 줄인다.

## 남은 과제

- 실제 기기 캡처 기준으로 360/390/430px별 픽셀 QA가 필요하다.
- 대형 캐릭터/몬스터 스프라이트 시트의 저해상도/고해상도 분리 로딩은 1.04 이후 핵심 과제다.
