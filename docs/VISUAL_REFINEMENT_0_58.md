# Alpha 0.58 Visual Refinement Pass

0.58은 0.57 품질 패스 이후의 추가 다듬기입니다. 목표는 “웹 모바일 RPG처럼 첫 화면, 마을, 필드, 전투, 전리품이 하나의 톤으로 이어지는가”입니다.

## 핵심 체크

- 타이틀: 0.58 애니풍 히어로 SVG와 마을 파노라마 레이어 보강
- 마을: NPC 클릭 시 짧은 대화 카드 출력 후 관련 메뉴로 연결
- 필드: 보스 게이트, 마력 안개, 보물 광원, 등불 아치 등 6종 소품 추가
- 전투: 일반 타격 과밀도 완화, 강타/희귀 드랍 집중 연출 유지
- 콘텐츠: 4개 후반 전선, 16개 메인 퀘스트, 5개 일일/주간 의뢰 추가
- 아이템: 신규 상자/재료/장비 8종 및 아이콘 추가

## QA 포인트

1. 모바일 세로 화면에서 NPC 대화 카드가 마을 주요 버튼을 가리지 않는지 확인
2. 보스 전선에서 필드 소품이 캐릭터/몬스터 클릭을 방해하지 않는지 확인
3. 희귀 드랍 팝업이 너무 오래 화면을 막지 않는지 확인
4. 가방 상세창의 신규 아이템 아이콘이 누락 없이 보이는지 확인
5. 긴 전선 목록에서 렌더링 지연이 과하지 않은지 확인
