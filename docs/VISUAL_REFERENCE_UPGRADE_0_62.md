# Alpha 0.62 Reference Visual Update Pass

## 목표

첨부 레퍼런스의 방향을 기준으로 `소울 온라인`의 시각 목표를 올린다.

- 밝은 하늘색 판타지 월드
- 아이보리 카드형 패널
- 블루 글래스 버튼
- 금장 테두리와 보석 포인트
- 귀여운 펫/몬스터와 선명한 캐릭터 실루엣
- 모바일 MMORPG식 풍부한 보상/알림/메뉴 밀도

## 이번 패치에서 실제 적용한 범위

### UI

- 타이틀, 로그인, 서버/캐릭터 선택, 마을, 필드 HUD에 0.62 스킨 추가
- 기존 0.61 WebP 패널/버튼/슬롯 텍스처를 재활용하되 색, 레이어, 그림자, 테두리를 레퍼런스 톤으로 조정
- 화폐/상태/전투 버튼/조이스틱/시트/상세카드에 통일된 블루·아이보리·금장 룩 적용

### 필드

- `addAlpha062ReferenceFieldPass()` 추가
- 스폰 포켓, 입구, 보스 후보 위치에 부드러운 광원과 경로 리본 추가
- 저급 신규 이미지를 생성하지 않고 Pixi 조명/프레임 레이어만 추가

### 캐릭터/몬스터

- 캐릭터 스프라이트 표시 스케일 상향
- 플레이어 바닥 원형 프레임, 클래스 오라, 네임플레이트 강화
- 몬스터/엘리트/보스에 베이스 플레어, 그림자, HP바, 이름 위치 조정

## 대규모 교체가 필요한 항목

이 레퍼런스 수준으로 완전히 가려면 단순 CSS 패치가 아니라 에셋 전면 패스가 필요하다.

1. 직업별 캐릭터 원화/전신/초상화/전투 스프라이트 재제작
2. 펫 3~5종 신규 스프라이트와 UI 아이콘 제작
3. 슬라임/늑대/고블린/곰/보스 몬스터의 귀여운 판타지 톤 재정의
4. 지형 타일을 전부 WebP 아틀라스로 통일
5. 마을/필드 배경을 지역별 키비주얼로 분리
6. 메뉴 아이콘 20종 이상 재제작
7. 보상 팝업, 레벨업, 획득 완료, 서버 선택, 퀘스트 패널 전용 일러스트 추가

## 품질 기준

- 임시 낙서형 PNG/SVG 추가 금지
- 저해상도 확대 사용 금지
- UI와 캐릭터의 광원 방향 불일치 금지
- 필드 스프라이트는 WebP 아틀라스 또는 검수된 PNG만 사용
- 기능 구현보다 먼저 화면 전체 톤 통일을 유지
