# Soul Online Alpha 0.50 Omega Overdrive Pass

0.50은 0.49 오메가 오버홀 이후의 두 번째 초대규모 업데이트입니다. 단순히 수치만 늘리는 패치가 아니라, 콘텐츠 총량, 초반 첫인상, 마을 비주얼, 후반 원정 루프, 진행 보드, 원정 보너스, 성능 규칙을 한꺼번에 확장했습니다.

## 핵심 변경

- 사냥터 48개에서 72개로 확장
- 메인 스토리 퀘스트 154개에서 274개로 확장
- 일일/순환 의뢰 117개에서 181개로 확장
- Lv.114~137 오메가 전선 24개 추가
- 레벨 상한 99에서 140으로 확장
- 신규 오메가 원정 재료 추가
  - 오메가 원정 증표
  - 별빛 재봉실
  - 공허의 기억
- 신규 후반 유물 추가
  - 고대 전투 깃발
  - 오메가 코어 유물
- 신규 카드 세트 추가
  - 오메가 선봉대
  - 악몽 사냥꾼
- 메인/보스/영혼 진행 기반 원정 지원 보너스 추가
- 누적 50킬 단위 원정 마일스톤 보상 추가
- 타이틀 화면 신규 애니풍 히어로 일러스트 적용
- 타이틀 배경/버튼 연출 강화
- 마을 파노라마 0.50 버전 추가
- 마을 NPC/오메가 수정/분위기 오브젝트 추가
- 오메가 지휘소 UI 추가
- 긴 목록 렌더링 안정화를 위해 기존 content-visibility 규칙 유지 및 오메가 전선 강조 규칙 추가
