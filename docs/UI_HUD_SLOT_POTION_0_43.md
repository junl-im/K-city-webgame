# Alpha 0.43 Lineage HUD + Slot Detail + Potion Balance Pass

## 목표

0.42 이후 사냥터 HUD가 다시 과밀해지고, 슬롯 UI가 지나치게 작아져 상세 정보 확인성이 떨어진 문제를 해결한다. 리니지 계열 모바일 MMORPG처럼 상단 캐릭터 바는 얇고 길게, 우측 하단 조작부는 개별 버튼 중심으로 정리한다.

## HUD 정리

- 상단 캐릭터 정보창을 화면 좌우 끝까지 쓰는 얇은 바 형태로 변경했다.
- 상단 메뉴 햄버거 버튼은 숨김 처리했다.
- 몬스터 타겟 정보창은 더 작게 줄이고 캐릭터 정보창 아래 중앙에 배치했다.
- 퀘스트 진행창은 우측 상단 작은 카드로 유지했다.
- 콤보 패널은 캐릭터 초상화 우측 아래에 작은 반투명 pill 형태로 이동했다.
- 우측 하단 액션 도크의 반투명 배경 패널을 제거하고 개별 원형 버튼만 남겼다.
- 스킬 버튼은 액션 버튼 위쪽에 같은 우측 하단 라인으로 배치했다.

## 슬롯 / 상세 정보 UX

- 가방, 카드, 스킬 슬롯의 아이콘 표시 면적을 키웠다.
- 슬롯 내부의 직접 장착/사용 버튼을 제거했다.
- 슬롯을 한 번 클릭하면 상세창이 열리고, 상세창 안에서 장착/해제/강화/사용을 선택한다.
- 모바일에서 hover 버튼이 안 보여서 기능을 못 찾는 문제를 줄였다.

## 물약 밸런스

- 하급 물약: 2% 회복
- 중급 물약: 3.5% 회복
- 상급 물약: 5% 회복
- 자동 물약은 약 0.95초 단위로 판단하고, 물약 사용에는 짧은 재사용 대기시간을 둔다.
- 회복량이 낮아진 만큼 상점 가격을 낮추고, 신규 캐릭터 기본 물약 수량을 늘렸다.
