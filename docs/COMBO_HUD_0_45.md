# Alpha 0.45 Combo HUD + Field Polish Pass

0.45는 사냥터 UI 피드백을 반영해 콤보 팝업 위치와 상단 HUD 겹침을 정리한 패치입니다.

## 핵심 수정

- 콤보 팝업을 캐릭터 정보창 아래가 아니라 전투 캐릭터 머리 우측 위 영역으로 이동
- 콤보 활성 중에만 반투명 팝업 표시
- `COMBO`, 보너스 수치, 남은 딜타임 표시 복구
- EXP 바를 최상단 독립 슬림바로 정리
- 캐릭터 정보 HUD를 EXP 바 아래로 내려 시각적 충돌 완화
- 상단 필드 햄버거 메뉴 버튼 마크업 제거
