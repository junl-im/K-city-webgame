# SOL Online Design Brief

## 목표

SOL Online은 Mir1의 저사양 2.5D 필드 사냥 감각에 카드 수집과 방치 성장을 섞은 모바일 웹 MMORPG입니다.

## 핵심 루프

1. 필드 접속
2. 이동 또는 자동사냥
3. 몬스터 처치
4. 경험치, 골드, 장비, 카드 획득
5. 카드 장착과 영혼 링크로 성장
6. 더 강한 지역과 월드보스 도전

## 전투

전투 흐름은 아래 순서입니다.

1. 공격 대상 확인
2. 거리 체크
3. 명중 체크
4. 공격력과 방어력 기반 피해 계산
5. 치명타 보정
6. HP 차감
7. 사망 판정
8. 드랍 판정

## 카드

카드는 몬스터 드랍과 이벤트 보상으로 획득합니다.

- N: 기본 생존 보정
- R: 이동, 방어, 공격속도
- SR: 직업 시너지
- SSR: 강한 복합 스탯

동일 카드가 쌓이면 자동 합성으로 레벨이 오릅니다.

## 영혼 링크

같은 몬스터를 일정 수 이상 처치하면 영혼이 해방됩니다. 카드는 확률 드랍이고 영혼은 확정 누적 성장이라, 운이 나빠도 플레이가 남습니다.

## 서버화 원칙

알파는 클라이언트 검증입니다. 서비스화 시 아래 항목은 Cloud Functions로 옮겨야 합니다.

- 전투 결과
- 드랍 결과
- 경험치 지급
- 재화 지급
- 카드 합성
- 거래소
- 월드보스 보상

## 아트 방향

- 2.5D 등각 타일
- 2.5~3등신 SD 캐릭터
- 명확한 외곽선
- 모바일에서 읽히는 큰 실루엣
- 카드 일러스트와 필드 스프라이트 분리
