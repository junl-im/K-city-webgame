# Alpha 0.63 Large Asset Upgrade Pass

## 목표

0.62에서 잡은 레퍼런스 방향을 실제 자산 단위로 확장한다. 이번 패치는 단순 CSS 색상 변경이 아니라 캐릭터, 몬스터, 지형, UI 소재, 아이콘, 카드/소울/아이템 자산을 같은 톤으로 맞추는 대규모 자산 업그레이드 1차 패스다.

## 적용 범위

### 캐릭터

- 직업/성별 6종 WebP 스프라이트 시트 재가공
- 밝은 하늘색/금장 레퍼런스에 맞춘 채도, 콘트라스트, 상단광, 림라이트 강화
- 필드에서 플레이어 옆에 미니 소울 펫 동행 실루엣 추가
- 0바이트/손상된 도사 남성 초상화 복구

### 몬스터/보스

- 슬라임, 늑대, 고블린, 가디언, 임프, 골렘, 망령, 드레이크, 하피, 기사, 필드보스, 용, 오크, 박쥐, 라바 골렘, 얼음 마녀, 로열가드, 리프트 비스트 18종 스프라이트 시트 재가공
- 종족별 포인트 컬러를 적용해 전투 중 실루엣과 이름표 가독성 강화
- 엘리트/보스 베이스 플레어와 0.63 필드 광원 패스 추가

### 지형/소품

- 8종 핵심 타일 WebP 소재감 재조정
- 나무, 바위, 횃불, 상자, 수정, 폐허 소품과 마을 건물 WebP 재가공
- 손상된 `prop-tree-05.webp` 복구
- 필드 입구, 보스 포켓, 사냥 루트에 0.63 빛결/결정/경로 소재감 추가

### UI/아이콘

- `ui-panel-glass-063.webp`, `ui-button-sky-063.webp`, `ui-slot-sky-063.webp`, `ui-location-pin-063.webp` 추가
- 타이틀/로그인/마을 키비주얼 063 버전 생성
- 메뉴 아이콘, 소울, 아이템, 카드, 효과 자산에 같은 프리미엄 후처리 톤 적용
- 손상된 `nightmare-robe.webp` 복구

## 코드 변경

- 앱 버전 `0.63.0`
- PWA 캐시 `soul-online-alpha-v0-63`
- `addAlpha063AssetUpgradeFieldPass()` 추가
- 필드 플레이어 루트에 `buildAlpha063Companion()` 펫 동행 레이어 추가
- 0.63 CSS 스킨 추가: 신규 UI 소재, 타이틀/로그인/마을 키비주얼, 슬롯/버튼/패널 재정의
- `asset-pack.json` 버전 및 0.63 적용 범위 갱신

## 품질 기준

- 저해상도 임시 이미지 추가 금지
- 기존 WebP 시트 크기와 프레임 메타 유지
- Pixi 애니메이션 메타 변경 없이 자산만 교체 가능하도록 유지
- 손상 파일 0바이트 금지
- 필드/마을/UI가 같은 하늘색·아이보리·금장 톤을 공유해야 함

- WebP 손상/0바이트 파일 검사
