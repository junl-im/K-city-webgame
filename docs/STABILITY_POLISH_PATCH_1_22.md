# Soul Online Alpha 1.22 안정화 다듬기 패치

## 목표

1.20 복구 커널과 1.21 다듬기 커널 위에서 첫 실행 안정성을 유지하면서, 남은 화면 전환 흔들림과 중복 보정 위험을 더 줄였습니다.

## 핵심 변경

- `src/ui/stabilityKernel122.ts` 추가
- `src/styles/alpha122.css` 추가
- START 터치 중복 입력 방지
- title / login / town / field 중 현재 장면 하나만 강제 표시
- 실행 당시 화면 크기 유지 보강
- 구형 화질 모드, 회전 가드, 펫/동행/룬/고스트 레이어 재등장 차단
- 필드 버튼 충돌 감지를 1.22에서도 유지하되 호출 빈도 제한
- System Doctor에 `1.22 안정화` 항목 추가
- PWA 캐시 `soul-online-alpha-v1-22` 갱신

## 의도

이번 패치는 새 기능을 넣기보다, 잘되던 게임을 흔들었던 누적 보정 구조를 더 안전하게 묶는 작업입니다. 구형 CSS 풀스택은 계속 기본 비활성이고, 현재 기본 경로는 1.20~1.22 복구/안정화 커널만 사용합니다.
