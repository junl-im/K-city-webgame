# Soul Online Alpha 1.00 - Hunting Field UI Overhaul

## 목표

0.99까지 남아 있던 사냥터 UI의 핵심 문제를 한 번에 정리한다.

- HUD, 퀘스트, 미니맵, 조이스틱, 스킬, 공격 버튼이 화면 밖으로 나가는 문제 방지
- 버튼이 서로 겹치거나 조이스틱과 스킬 버튼이 충돌하는 문제 완화
- 투톤 블루/골드 느낌을 줄이고 파치먼트, 아쿠아, 바이올렛, 로즈, 골드가 섞인 원화풍 UI로 변경
- 저사양 모바일에서 렉을 줄이기 위한 렌더 해상도, FPS, 텍스처 로딩, 이펙트 밀도 조정

## 추가 파일

- `src/styles/alpha100.css`
- `src/ui/fieldHudOverhaul100.ts`
- `public/assets/ui/fantasy/100/*.webp`

## 주요 변경

### 1. 사냥터 HUD 단일 셸

기존 body/root에 흩어져 있던 사냥터 UI를 런타임에서 `.field-hud-shell-100` 안으로 모아 관리한다.
이동 대상은 다음과 같다.

- `.hud-top`
- `.resource-strip`
- `.target-card`
- `.field-quest-tracker`
- `.field-chain-meter`
- `.field-minimap`
- `.combat-log`
- `.joystick`
- `.potion-dock`
- `.skill-dock`
- `.action-dock`

이 방식은 이벤트 리스너를 유지하면서 레이아웃을 한 곳에서 통제하기 위한 조치다.

### 2. 모바일 배치 원칙

- 상단 좌측: 캐릭터 HUD
- 상단 우측: 미니맵
- 미니맵 아래/좌측: 진행 퀘스트
- 하단 좌측: 조이스틱
- 하단 중앙 좌측: HP/MP 물약
- 하단 우측: 공격/자동/더보기
- 공격 버튼 위: 스킬 3개

360px 폭에서도 스킬/공격/조이스틱이 서로 겹치지 않도록 미디어 쿼리를 추가했다.

### 3. 성능 최적화

- 저사양/데이터 절약/라이트 모드에서 `soul-online-field-lite-100` 사용
- Pixi 렌더 해상도 하향
- 저/중간 사양에서 안티앨리어싱 비활성화
- 저사양 FPS 40, 중간 사양 FPS 50 제한
- 텍스처 로딩 병렬 수 축소
- 저사양에서 이펙트 텍스트 과밀 생성 억제
- 몬스터 밀도 스케일 소폭 하향
