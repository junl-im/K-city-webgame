# Soul Online alpha 1.01 - Performance/Game Polish Pass

목표는 사냥터 렉과 모바일 조작 안정성 개선이다. 1.00에서 사냥터 HUD를 새로 잡았고, 1.01에서는 필드 렌더링 비용 자체를 줄였다.

## 변경 요약

- `src/ui/performanceTuner101.ts` 추가: 기기 메모리/코어/네트워크/데이터 절약 모드를 기준으로 lite/balanced/quality 렌더 계층을 자동 선택한다.
- `src/styles/alpha101.css` 추가: 필드 HUD, 조작 버튼, 사냥터 패널의 GPU 부담을 낮추고 저사양에서 blur/장식/애니메이션을 줄인다.
- `src/game/SolGame.ts` 최적화:
  - 저사양 렌더 해상도 0.85, FPS 34 제한.
  - 중간 사양 렌더 해상도 1.05, FPS 45 제한.
  - 텍스처 로딩 병렬 수 축소.
  - 라이트/균형 모드에서 과거 필드 장식 패스를 건너뜀.
  - 화면 밖 몬스터 업데이트 빈도 축소.
  - 엔티티 정렬과 HUD emit 빈도 제한.
  - 이펙트/데미지 텍스트/먼지/화면 흔들림 생성량 제한.
- System Doctor에 `1.01 성능` 타일 추가.
- PWA cache를 `soul-online-alpha-v1-01`로 갱신.

## 남은 과제

대형 캐릭터/몬스터 스프라이트 시트는 아직 원본 크기가 크다. 다음 단계는 실제 저해상도 스프라이트 시트를 생성해서 기기 등급별로 로딩하는 것이다.
