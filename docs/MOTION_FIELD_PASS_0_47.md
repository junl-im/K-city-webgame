# 0.47 Motion + Field Object Pass

## 목표

사냥터 화면이 정적인 느낌에서 벗어나도록 전투 모션, 스킬 이펙트, 필드 오브젝트 밀도를 한 번에 보강했다.

## 적용 내용

- `public/assets/soulpack/effects`의 이펙트 이미지를 번들 fallback으로 복사해 Vite 빌드에 포함했다.
- Pixi 전투 레이어에 soul slash / fireball / lightning / holy nova / dark rift 스프라이트를 합성했다.
- 캐릭터 공격/시전 시 afterimage를 표시해 실제 2.5D 시트 모션을 더 강하게 보이게 했다.
- 이동 중 발밑 먼지 이펙트를 추가했다.
- 몬스터 사망 시 영혼 방출 효과를 추가했다.
- 신규 SVG 필드 장식 7종을 추가하고 지역별로 배치했다.
- 지역별 마력 입자/빛무리 레이어를 추가했다.

## 주의

- UI 구조와 저장 데이터는 변경하지 않았다.
- 장식 오브젝트는 충돌 판정을 추가하지 않아 자동사냥 경로에는 영향을 주지 않는다.
- 이펙트는 Pixi `fxLayer`에만 생성되고 짧은 애니메이션 후 제거된다.
