# Soul Online Alpha 0.65 비주얼 폴리시 패치

0.64에서 이어지는 가독성/시인성/터치 안정화 패치입니다.

## 주요 목표

- 0.60 시작 화면 키비주얼 유지
- TOUCH TO START 버튼을 모바일 화면 중앙 라인으로 재배치
- 밝은 하늘/아이보리/블루/금장 UI에 맞춘 텍스트 색상 계층 재정리
- 클릭/터치가 장식 레이어에 막히지 않도록 포인터 이벤트 보강
- 버튼/핫스팟/전투 액션에 과하지 않은 모션 피드백 추가
- 한국어 가독성을 위해 글자 간격, 숫자 정렬, 줄 간격 보정

## 적용 방식

GitHub Desktop 로컬 저장소 루트에 압축을 풀고 같은 파일은 덮어쓰기합니다.
그 뒤 `npm install`, `npm run build`로 확인합니다.
