# 0.59 Deep Polish Pass

목표: 타게임급 모바일 RPG에 가까워지도록 타이틀, 마을, 필드, 전투, 가방, 전리품 흐름을 한 번 더 다듬는다.

- 타이틀: 0.59 애니풍 히어로 SVG와 마을 파노라마 배경 강화
- 마을: 파노라마 광원, NPC/기능 패널과 어울리는 고급 판타지 톤 유지
- 필드: 왕실 아치, 전쟁 깃발, 촛불 원, 대리석 균열, 영혼 분수, 사냥 표식 추가
- 전투: 타격감 과밀도는 억제하고 희귀/강타/보스 연출은 더 선명하게 유지
- 가방: 아이콘 시인성, 글자 크기, 상세창 흐름 유지
- 드랍: 사냥 허가증, 타격 연마유, 수선실, 광택석, 보급함, 신규 장비 드랍 루프 추가
