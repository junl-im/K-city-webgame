# Alpha 0.82 UI Polish Notes

## 목적

0.82는 게임명을 **Soul Online / 소울 온라인** 기준으로 재확인하고, 0.81에서 도입한 판타지 UI 스킨을 실제 플레이 흐름에 맞게 더 다듬는 패치다.

이전 패치 ZIP 이름에 남아 있던 `K-city`는 업로드된 원본 폴더명에서 이어진 작업명일 뿐이며, 프로젝트 내부의 앱 이름과 사용자 노출 브랜드는 `소울 온라인`으로 유지한다.

## 적용 내용

- 패키지 버전 `0.82.0`으로 갱신
- PWA 캐시 `soul-online-alpha-v0-82`로 갱신
- `fantasy-ui-082`, `title-screen-082`, `login-screen-082`, `town-screen-082` 스킨 레이어 추가
- 마을 드로어 상단에 `town-flow-advisor-082` 진행 가이드 추가
  - 현재 스토리 목표
  - 진행도 바
  - 추천 사냥터 이동
  - 전투력/보상 대기/가방 사용량 표시
- 모바일 safe-frame 강화
  - `100dvh`/`100svh` 기준 화면 잠금
  - 가로 오버플로 차단
  - 작은 화면에서 일부 보조 UI 자동 축소/숨김
- 마을 로비 버튼, 지갑, 하단 메뉴, 드로어, 상점/가방/스토리 행 스타일 보강
- 필드 HUD/미니맵/전투 로그/조이스틱/스킬 독 충돌 완화

## UX 기준

0.82부터 마을 드로어는 단순 메뉴가 아니라 “현재 무엇을 해야 하는지”를 보여주는 진행 허브로 사용한다.

우선순위는 다음과 같다.

1. 현재 스토리 목표 확인
2. 추천 사냥터로 즉시 이동
3. 장비/가방 정비
4. 일일 보상 회수
5. 상점/스킬/카드/혈맹으로 확장

0.83에서는 가방, 스킬, 상점, 보상 팝업의 내부 구조를 더 강하게 통일해야 한다.
특히 `styles.css`가 계속 커지고 있으므로, 다음부터는 UI 영역별 CSS 분리 작업도 같이 진행하는 편이 좋다.
