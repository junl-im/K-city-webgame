# 0.29 Item Compare + Loot Presentation Pass

## 장비 비교

아이템 상세 팝업에서 현재 장착 장비와 후보 장비의 스탯을 즉시 비교합니다. 장착 중인 장비를 다시 누르면 해제 시 손실도 표시됩니다.

## 획득 연출

상점 구매, 스킬 습득, 필드 아이템/카드/소울젬 드랍은 화면 전면의 loot-pop 연출로 표시됩니다. 희귀도에 따라 색상과 광원이 달라집니다.
