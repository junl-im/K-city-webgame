# Alpha 0.51 Loot & Lawful Overhaul

## 목표

아이템 파밍의 다양성, 가방 분류, 강화 시스템 가시성, 아이템 아이콘 정합성, 필드 오브젝트 품질, 라우풀 성향 루프를 한 번에 개선한다.

1. 신규 장비/재료 추가
2. 몬스터별 추가 드랍 오버레이 적용
3. 모든 아이템용 WebP 아이콘 생성
4. 인벤토리를 장비/소모품/재료/스킬서로 분리
5. 강화 공방 패널 추가
6. 라우풀 성향 수치 저장 및 사냥 보상 연결
7. CHAIN BONUS 표기 복구
8. 필드 오브젝트 그림자/광원/스케일 보정
