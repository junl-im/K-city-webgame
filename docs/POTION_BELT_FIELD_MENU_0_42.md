# Soul Online Alpha 0.42 - Potion Belt + Field Menu Pass

## 목적

0.41에서 물약이 들어갔지만 필드 단축 버튼이 하급 물약에 고정되어 있었다. 0.42는 전투 중 현재 상태와 보유 물약을 기준으로 적절한 물약 등급을 자동 선택하는 벨트 구조를 추가한다.

## 주요 변경

- 필드 메뉴 버튼 중복 ID 제거
- HP/MP 물약 단축 버튼을 자동 등급 선택 방식으로 변경
- 자동사냥 물약 사용 로직을 하급 고정에서 하급/중급/상급 선택으로 변경
- 상급 생명 물약, 상급 마나 물약 추가
- 사냥터 난이도별 물약 드랍 추가
- 마을 상점 5개 묶음 구매 추가
- 인벤토리/상점에 물약 벨트 요약 카드 추가

## 테스트 포인트

1. 필드 우측 하단 메뉴 버튼이 정상적으로 캐릭터/계정 시트를 여는지 확인
2. HP가 낮을 때 HP 단축 버튼이 보유 물약 중 상황에 맞는 물약을 사용하는지 확인
3. 자동사냥 중 HP/MP가 낮아졌을 때 중급/상급 물약까지 사용하는지 확인
4. 마을 상점에서 5개 묶음 구매 후 수량과 골드가 정상 반영되는지 확인
