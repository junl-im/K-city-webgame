# Soul Online 0.23 Real Visual Asset Expansion

0.23 패치는 미르, 뮤, 리니지, 오딘, 와우 계열의 어두운 판타지 2.5D 감성을 참고하되, 상용 게임 리소스를 복제하지 않고 소울 온라인용 오리지널 절차형 고해상도 샘플 리소스로 구성했습니다.

## 캐릭터 시트

공통 규격:

- 8방향: S, SW, W, NW, N, NE, E, SE
- 프레임: 160x240
- 모션 구간: 대기, 걷기, 달리기, 공격, 피격, 사망, 스킬 시전
- 한 시트당 48프레임 x 8방향

포함 시트:

- 검혼 남자 / 여자
- 영술사 남자 / 여자
- 성휘사 남자 / 여자

경로:

- `public/assets/soulpack/characters/`
- `src/assets/2p5d/characters/`

## 몬스터 시트

공통 규격:

- 프레임: 160x200
- 대기/이동/공격/피격/사망/스킬형 행위 연출 대응

포함 몬스터:

- 슬라임

- 고블린
- 그림자 임프
- 이끼 골렘

- 흑수정 곰
- 필드 보스
- 드래곤

## 0.23 UI 수정 요약

- 타이틀/로그인/서버/캐릭터 화면에서는 전투 HUD와 마을 메뉴가 보이지 않도록 수정
- 마을에서는 하단 5버튼 메뉴만 사용
- 사냥터에서는 마을 하단 메뉴를 숨기고 전투 HUD, 조이스틱, 자동/스킬/마을복귀만 표시
- 진행중 퀘스트를 사냥터 HUD에 표시
- 사냥터 선택을 배너형에서 월드맵형 그래픽 카드로 정리
- 캐릭터 이동 속도를 이전보다 안정적으로 낮춤

## 다음 단계

이번 리소스는 게임에 바로 들어가는 고해상도 샘플팩입니다. 진짜 상용급 퀄리티로 더 올리려면 같은 파일명과 시트 규격을 유지한 채 손그림/3D 프리렌더/외주 원화 기반 리소스로 교체하면 됩니다.
