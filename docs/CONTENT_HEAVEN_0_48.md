# 0.48 Content Heaven + Anime Title + RPG Town Pass

0.48은 콘텐츠 총량과 첫인상을 크게 올리는 대형 업데이트입니다.

## 콘텐츠 확장

- 사냥터: 28개
- 메인 스토리: 93개
- 일일/순환 의뢰: 77개

후반부는 Lv.34부터 Lv.72까지 이어지는 장기 성장 구간으로 확장했습니다. 보스 누적 토벌, 드레이크/심연룡 장기 추적, 장비 공명과 스킬 숙련을 요구하는 루프를 추가했습니다.
