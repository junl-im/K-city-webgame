# Alpha 0.53 Blood Pledge Citadel Pass

## 목표

0.52의 라우풀/카오틱, 보스 상자, 고급 디자인 패스 위에 혈맹형 장기 성장 루프와 NPC 계약 의뢰를 추가했습니다.

## 핵심 변경

- 혈맹 저장 구조 추가
- 혈맹 레벨/경험치/공헌도/문장/기부 기록 저장
- 사냥 처치 시 혈맹 경험치와 공헌도 획득
- 혈맹 레벨별 HP/MP/공격/방어/드랍 보너스 적용
- 마을 혈맹 지휘소 UI 추가
- 골드/공헌 주화/군주의 인장 기부 추가
- 혈맹 전쟁 궤짝 교환 추가
- 마을 NPC 계약 의뢰 3종 추가
- 혈맹 전선 사냥터 6개 추가
- 메인 스토리 퀘스트 24개 추가
- 일일/주간 의뢰 6개 추가
- 신규 아이템 8종 및 아이콘 8종 추가
- 혈맹 성채 전선용 필드 장식/광원 추가
- 타이틀 히어로 SVG, 마을 파노라마 SVG 0.53 버전 추가
