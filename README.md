# 소울 온라인 Alpha 0.27 Inventory/Card/Skill UI & 2.5D Field Asset Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.27은 슬롯형 UI를 다시 정리하고, 가방/카드/스킬/영혼 상세 팝업과 필드 타일/오브젝트 2.5D 리소스 개선을 적용한 버전입니다.

## 0.27 변경점

- 가방/카드/스킬 슬롯 UI 재정리
  - 슬롯을 정사각형 기준으로 재배치
  - 카드/가방/스킬 칸이 줄어들어도 아이콘이 찌그러지지 않도록 수정
  - 모바일에서는 카드 3열, 가방 4열, 스킬 3열 중심으로 표시
- 슬롯 상세 팝업 추가
  - 카드 슬롯 클릭 시 카드 이미지, 등급, 레벨, 효과, 장착 버튼 표시
  - 장비/재료/스킬북 클릭 시 상세, 능력치, 강화 확률/비용, 장착/강화 버튼 표시
  - 스킬 클릭 시 MP, 쿨타임, 사거리, 범위, 습득 상태 표시
  - 영혼 클릭 시 해방 조건과 효과 표시
- 가방 아이콘 개선
  - 텍스트만 보이던 아이템 슬롯을 무기/방어구/유물/재료/스킬서 타입별 아이콘 패널로 변경
- 필드 2.5D 타일/오브젝트 개선
  - 잔디/흙길/이끼/석재/수정/물/절벽/포탈 타일을 등각 2.5D WebP 스타일로 재생성
  - 나무 10종, 바위 10종, 상자 5종, 횃불 5종을 입체감 있는 WebP 오브젝트로 재생성
  - 수정, 폐허 프랍도 입체감 강화
- 버전 갱신
  - 앱 버전: `0.27.0`
  - 저장 버전: `SAVE_VERSION = 23`
  - PWA 캐시: `soul-online-alpha-v0-27`
