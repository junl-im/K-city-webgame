# 소울 온라인 Alpha 0.40 Gear Resonance Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.40은 0.39 Skill Mastery Pass 위에 장비 공명과 중복 장비 분해 루프를 추가한 개발 패치입니다.

## 0.40 변경점

- 장비 공명 시스템 추가
  - 무기, 방어구, 유물을 모두 장착하면 기본 공명 발동
  - 세 부위 최저 등급에 따라 R / SR / SSR / UR 공명 발동
  - 세 부위 최저 강화 단계에 따라 +5 / +10 / +15 / +20 공명 발동
- 장비 공명 보너스가 실제 전투 능력치에 반영
- 마을/필드 인벤토리에 장비 공명 요약 패널 추가
- 캐릭터 정보창에 활성 장비 공명 요약 표시
- 마을 인벤토리에 중복 장비 분해 버튼 추가
  - 각 장비는 최소 1개 보존
  - 초과 수량만 골드, 소울 파편, 강화석으로 전환
- 버전, PWA 캐시, 에셋팩 메타데이터를 0.40 기준으로 정리

## 실행

```bash
npm ci
npm run build
npm run dev
```
