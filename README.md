# 소울 온라인 Alpha 0.45 Combo HUD + Field Polish Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.45는 0.44 오프닝 패치 이후 사냥터 HUD를 다시 정리하고, 콤보 팝업을 캐릭터 머리 우측 위 전투 위치로 복구한 개발 패치입니다.

## 0.45 변경점

- 콤보 팝업을 캐릭터 정보바 안이 아니라 중앙 캐릭터 머리 우측 위 위치로 이동했습니다.
- 콤보 문구를 `COMBO` 중심으로 정리했습니다.
- 콤보 혜택, 남은 시간, 딜타임 안내를 다시 표시합니다.
- 콤보가 없을 때는 팝업이 전투 화면에 떠 있지 않도록 숨김 처리했습니다.
- 최상단 EXP 바와 캐릭터 정보 HUD가 겹쳐 보이지 않도록 분리했습니다.
- 필드 캐릭터 HUD를 EXP 독립바 아래로 확실히 내렸습니다.
- 상단에 남아 있던 필드 햄버거 메뉴 버튼 마크업을 제거했습니다.
- 몬스터 타겟창과 퀘스트 진행창 위치를 새 HUD 기준에 맞게 보정했습니다.

## 빌드 정보

- 앱 버전: `0.45.0`
- 저장 버전: `SAVE_VERSION = 31` 유지
- PWA 캐시: `soul-online-alpha-v0-45`
- 빌드 명령: `npm run build`
