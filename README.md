# 소울 온라인 Alpha 0.46 Auto Tactics + Boss Bounty Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.46은 0.45 전투 HUD 안정화 이후 자동사냥 세부 설정과 보스 토벌 보상 루프를 추가한 개발 패치입니다.

## 0.46 변경점

- 자동사냥 세부 설정 추가
  - 자동 스킬 ON/OFF
  - HP 물약 자동 사용 ON/OFF
  - MP 물약 자동 사용 ON/OFF
  - HP/MP 물약 발동 기준 선택
  - 보스 우선 타겟팅 ON/OFF
- 필드 계정/설정 시트와 마을 설정/보스 메뉴에 자동사냥 설정 패널 추가
- 필드보스/심연룡이 `균열 토벌 훈장`을 드랍하도록 추가
- 마을 보스 메뉴에서 훈장을 보급 상자/정예 상자로 교환 가능
- 보스 메뉴 UI를 현상금 루프 중심으로 개편
- 저장 버전 `SAVE_VERSION = 32`
- 앱 버전: `0.46.0`
- PWA 캐시: `soul-online-alpha-v0-46`
