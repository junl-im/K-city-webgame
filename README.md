# 소울 온라인 Alpha 0.29 Item Compare & Loot Presentation Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.29는 장비 비교, 드랍 연출, 구매 획득 피드백, 강화/장착 체감 표시를 보강한 버전입니다.

## 0.29 변경점

- 장비 상세 팝업에 현재 장착 장비 대비 스탯 비교 추가
- HP/MP/공격/방어/공속/치명/이속 및 전투력 증가·감소 표시
- 장착 중 장비는 해제 시 변화량 비교 표시
- 상점 구매 시 아이템/스킬 습득 획득 연출 추가
- 필드 드랍 시 카드/아이템/소울젬 획득 팝업 추가
- 희귀도별 획득 팝업 색상/광원 차등 적용
- 상세 팝업 이미지/능력치/강화 정보 구역 정리

## 버전

- 앱 버전: `0.29.0`
- 저장 버전: `SAVE_VERSION = 25`
- PWA 캐시: `soul-online-alpha-v0-29`
