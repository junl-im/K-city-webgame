# 소울 온라인 Alpha 0.35 UI Stabilization Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.35는 0.32 그래픽 UI 스킨과 0.33 필드 패치 이후 흐트러진 UI 배치를 먼저 안정화하는 수정 패치입니다.

## 0.35 변경점

- 마을 사냥터 패널의 잘못된 HTML 닫힘 구조 수정
- 필드 HUD, 몬스터 타겟, 퀘스트 트래커 겹침 방지
- 미니맵 제거 정책 고정
- 모바일/초소형 화면 전용 HUD 배치 정리
- 스킬 도크와 전투 액션 도크 크기/위치 재조정
- 시트, 마을 드로어, 하단 메뉴 z-index와 스크롤 안정화
- 카드/인벤토리/스킬 슬롯 크기 재조정
- Wenrexa 그래픽 UI 스킨은 유지하되 과한 전역 적용을 마지막 안정화 규칙으로 완화
- 버전, PWA 캐시, 에셋팩 메타데이터를 0.35 기준으로 정리

## 유지 내용

- 0.33 사냥터 자동 추천 가이드
- Infernus 고레벨 필드 오브젝트 패스
- 0.32 그래픽 UI 에셋 스킨
- 0.31 퀘스트 대폭 확장, 미니맵 제거 방향, 슬롯 개선 방향

## 개발 명령

```bash
npm ci

npm run dev
```
