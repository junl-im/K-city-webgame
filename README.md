# Soul Online Alpha 0.89 덮어쓰기 패치

Soul Online 0.89는 0.88의 가방/상점 확인 UX 위에 스킬 성장 화면, 숙련 강화 확인, 에셋 예열 우선순위, 기술 점검을 더하는 안정화 패치입니다. 0.88 위에 그대로 덮어쓰는 소스 패치로 사용할 수 있습니다.

## 적용 순서

1. GitHub Desktop에서 현재 상태를 먼저 커밋합니다.
2. 이 ZIP을 프로젝트 루트에 압축 해제해 덮어씁니다.
3. `npm install`을 실행합니다.
4. `npm run build`로 빌드 확인을 합니다.
5. Firebase Hosting 사용 시 `firebase deploy --only hosting`으로 배포합니다.

## 0.89 핵심

- 버전을 `0.89.0`으로 갱신했습니다.
- PWA 캐시를 `soul-online-alpha-v0-89`로 갱신했습니다.
- `src/ui/skillPanelRenderer089.ts`를 추가해 스킬 성장 상태/강화 확인 UI 렌더링을 분리했습니다.
- `src/ui/assetPreload089.ts`를 추가해 현재 직업/화면 상태에 맞춘 에셋 예열 우선순위를 만들었습니다.
- `src/styles/alpha089.css`를 추가해 0.89 전용 스킬 확인 패널, 스킬 연결성 스트립, safe-frame 보정을 분리했습니다.
- 스킬 숙련 강화는 즉시 소비가 아니라 확인 패널을 거치도록 변경했습니다.
- 스킬 화면에 습득/사용 가능/강화 가능/숙련 합계 점검 스트립을 추가했습니다.
- System Doctor/Tech Health에 스킬 성장 상태와 에셋 예열 계획을 추가했습니다.
- UI 안전 영역 검사에 스킬 강화 확인 패널과 스킬 연결성 스트립을 포함했습니다.

## 계속 남은 과제

- `main.ts`는 아직 큽니다. 0.90부터 카드/소울/보상 렌더러를 더 분리해야 합니다.
- `styles.css`는 아직 거대합니다. 신규 스타일은 계속 별도 CSS로 분리해야 합니다.
- 대형 스프라이트 시트가 모바일 첫 로딩을 압박하므로, 다음 단계에서는 캐릭터/몬스터 시트의 지연 로딩과 해상도 분기를 더 봐야 합니다.
