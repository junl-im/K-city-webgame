# Soul Online Alpha 0.64 → 0.65 GitHub Desktop 덮어쓰기 패치

이미 0.64까지 적용한 프로젝트에 바로 덮어쓰는 소형 패치입니다.

## 적용 순서

1. GitHub Desktop에서 현재 상태를 커밋합니다.
2. 이 ZIP을 압축 해제합니다.
3. 압축 안의 파일들을 프로젝트 루트에 그대로 복사하고 같은 파일은 덮어씁니다.
4. `npm install`, `npm run build`로 확인합니다.

## 핵심 변경

- 시작 화면 터치 버튼 위치 중앙 보정
- 밝은 UI용 글씨색/그림자/패널 대비 보정
- 터치 레이어 충돌 방지 보강
- 버튼/핫스팟/공격 버튼 모션 피드백 추가
- PWA 캐시 `soul-online-alpha-v0-65` 갱신
