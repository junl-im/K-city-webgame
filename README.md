# 소울 온라인 Alpha 0.14 Map Structure Pass

모바일 세로형 2.5D 웹 MMORPG 프로토타입입니다. 0.14에서는 모바일 마을 레이아웃 오버플로우를 보정하고, 접속 단계 탭 표시를 제거했으며, 사냥터별 독립 지형/진입로/몬스터 배치와 필드 미니맵 HUD를 추가했습니다.

## 실행

```bash
npm ci
npm run dev
```

## 빌드

```bash
npm run build
```

## 0.14 Map Structure Pass

- 모바일 마을 화면 오버플로우 보정
  - `100dvh` 내부 스크롤 고정
  - 퀘스트/보상 패널이 화면을 밀어내지 않도록 드로어 높이 제한
  - 마을 디오라마와 상단 재화 영역 축소
- 접속 화면 단계 탭 제거
  - 로그인/서버/캐릭터/입장 탭 표시 제거
  - 현재 단계 안내 문구만 남기는 자연스러운 플로우로 변경
- 사냥터 진입 안정화
  - 입장 실패 시 마을로 복구하고 오류 토스트 표시
  - 사냥터 시작 좌표와 HUD 갱신 보강
- 0.14 맵 구조 패스
  - 사냥터별 독립 지형 생성
  - 초록 숲/수정 이끼길/고블린 길목/흑수정 동굴/수정 레이드 터의 바닥 구성을 분리
  - 안전 진입로와 몬스터 구역 라벨 추가
  - 사냥터별 몬스터 스폰 후보 분리
  - 필드 미니맵 HUD 추가

## 이전 주요 기능 유지

- 첫 화면 → 로그인 → 서버 → 캐릭터 → 마을 입장 플로우
- 루미나 마을 허브
- 스토리 퀘스트
- 사냥터 지역 진행도
- 몬스터 AI 상태머신
- 스킬 버튼/쿨타임/MP 소모/범위 공격
- 카드 세트 효과
- 장비 강화
- BGM/SFX 및 사운드 설정
- PWA 세로 모드 설정
