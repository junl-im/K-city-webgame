# 소울 온라인 Alpha 0.28 UI Feedback & 2.5D Polish Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.28은 구매/강화/장착 피드백, 스킬서 품절 처리, 사냥터 HUD 재배치, 인게임 종료 팝업, 슬롯 상세 팝업 전면 표시, 2.5D 아이템/스킬/카드/영혼/지형 리소스 보강을 적용한 버전입니다.

## 0.28 변경점

- 구매/강화/장착/주요 버튼 클릭 피드백 이펙트 추가
- 스킬서는 구매 후 품절 처리, 중복 구매 방지
- EXP 바 최상단 고정
- 타겟 몬스터 정보 상단 중앙 소형 카드화
- 진행중 퀘스트 클릭 시 보상 수령 또는 다음 퀘스트 사냥터 자동 이동
- 미니맵을 프로필 아래로 이동
- 골드/소울젬은 가방 패널에서 확인
- 캐릭터 정보와 환경설정/계정 저장 카드를 분리
- 모바일 뒤로가기 종료 확인을 브라우저 팝업이 아닌 인게임 팝업으로 변경
- 상세 팝업이 뒤로 깔리는 문제 수정
- 슬롯 크기와 표시 정보 재조정
- 장비/스킬/영혼 이미지 추가, 카드 이미지 재생성
- 업로드 지형 레퍼런스를 참고한 2.5D 바위/나무/상자/횃불 오브젝트 보강
