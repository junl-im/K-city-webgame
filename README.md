# 소울 온라인 Alpha 0.12 Audio & Visual Pass

모바일 세로형 2.5D 웹 MMORPG 프로토타입입니다. 0.11과 0.12를 한 번에 묶어 **BGM/효과음 시스템**, **사운드 온오프 설정**, **필드/마을 장면별 사운드 전환**, **2.5D 타일 비주얼 보강**, **필드 분위기 레이어**를 적용했습니다.

## 0.11 Audio Pass

- Web Audio 기반 임시 BGM 엔진 추가
  - 타이틀/마을/필드/보스 장면별 분위기 전환
  - 모바일 브라우저 정책에 맞춰 `게임 시작` 같은 사용자 입력 후 오디오 잠금 해제
- 효과음 추가
  - 버튼 클릭
  - 자동사냥 시작/정지
  - 일반 공격
  - 피격
  - 스킬 사용
  - 회복
  - 아이템/카드/소울젬 보상
  - 장비 강화
  - 레벨업
  - 기절/오류
- 사운드 설정 UI 추가
  - 타이틀 화면 BGM 토글
  - 마을 빠른 BGM 토글
  - 마을 `사운드` 패널
  - 필드 계정 패널 내 사운드 설정
- 설정 저장
  - BGM ON/OFF
  - 효과음 ON/OFF
  - localStorage에 전역 저장

## 0.12 Visual Pass

- 필드 타일 SVG 개선
  - 잔디 타일에 풀결/흙빛/입체 엣지 추가
  - 석재 타일에 균열/판석 질감 추가
  - 물 타일에 파도선/반사 하이라이트 추가
  - 포탈 타일에 발광 룬/중앙 게이트 강화
- 필드 분위기 레이어 추가
  - 사냥터별 색감 미스트
  - 바닥 위 영혼 광맥 라인
  - 보스/동굴/숲 지역별 다른 색조
- 마을 비주얼 보강
  - 건물 소울등 발광 장식
  - 마을 배경 하이라이트 레이어
- PWA 캐시 버전 갱신
  - `soul-online-alpha-v0-12`

## 이전 버전 주요 기능 유지

- 첫 화면 → 로그인 → 서버 → 캐릭터 → 마을 입장 흐름
- 캐릭터 최대 4개 생성/삭제/접속
- 단일 서버: 곰같은여우 서버
- 루미나 마을 허브
- 스토리 퀘스트
- 일일 의뢰
- 지역 진행도/사냥터 해금
- 몬스터 AI 상태머신
- 스킬 버튼/쿨타임/MP 소모/범위 공격
- 카드 세트 효과
- 장비 장착/해제/강화
- 세로모드 고정 보강
- 자동사냥 끼임 복구 로직

```bash
npm install

```

```bash
npm run build
```

## 저장

- 로컬 저장은 브라우저 localStorage를 사용합니다.
- Firebase 설정이 있으면 `/users/{uid}`와 `/rankings/{uid}`에 클라우드 저장을 시도합니다.
- 클라우드 저장 실패 시 로컬 저장은 유지되고, 자동 저장 실패 메시지는 반복 노출되지 않습니다.
