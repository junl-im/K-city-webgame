# 소울 온라인 Alpha 0.30 External Asset Integration Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.30은 사용자가 제공한 외부 에셋팩을 분석하고, 실제 게임 런타임 에셋 구조에 맞춰 타일/오브젝트/아이템/스킬/영혼/카드/타이틀 비주얼을 교체한 버전입니다.

## 0.30 변경점

- PVGames Infernus Free 계열 오브젝트/지형 질감 일부 반영
- LOOT PIXELS Adventure Pack 아이콘 계열을 장비/아이템 프레임 이미지에 반영
- Elven Forest/Basic Dungeon 계열 시트를 참고해 마을 건물, 지형 질감, 카드 배경 보강
- 사냥터 바닥 타일 8종을 WebP 등각 2.5D 스타일로 재생성
- 나무/바위/상자/횃불/수정/폐허 오브젝트를 외부 에셋 기반으로 재구성
- 장비/스킬/영혼 아이콘을 256x256 프레임 이미지로 보강
- 카드 이미지를 몬스터/캐릭터 시트와 신규 배경 프레임으로 재생성
- 타이틀 배경과 타이틀 캐릭터 이미지를 외부 에셋 톤에 맞춰 재구성
- 외부 에셋 출처와 주의사항 문서 추가

## 실행

```bash
npm ci --no-audit --no-fund
npm run build
```

## 버전

- 앱 버전: `0.30.0`
- 저장 버전: `SAVE_VERSION = 26`
- PWA 캐시: `soul-online-alpha-v0-30`

## 에셋 사용 주의

이번 버전에는 사용자가 업로드한 외부/무료 에셋팩에서 변환한 런타임 리소스가 포함됩니다. 공개 배포나 상업 이용 전에는 각 원본 에셋팩의 라이선스를 반드시 다시 확인하세요.
