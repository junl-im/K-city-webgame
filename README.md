# 소울 온라인 Alpha 0.31 UI / Quest Expansion Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.31은 사냥터 HUD 겹침, 미니맵 위치 문제, 과한 확인 팝업, 슬롯 이미지 표시 문제를 정리하고 퀘스트 볼륨과 필드 오브젝트 배치를 크게 보강한 버전입니다.

## 0.31 변경점

- 사냥터에서 플레이어 HP/MP 카드와 몬스터 타겟 카드가 겹치지 않도록 HUD 배치 재정렬
- 위치가 불안정하고 시야를 방해하던 미니맵 UI 제거
- 퀘스트 스토리 46개, 일일/순환 퀘스트 30개로 대폭 확장
- 초반부터 후반 지역까지 이어지는 장기 누적 사냥/성장 목표 추가
- 필드 바닥 오브젝트를 지역별 큐레이션 방식으로 재배치해 난잡한 장식 밀도 완화
- 버튼 클릭마다 뜨던 중앙 확인 피드백 팝업 제거
- 인벤토리, 스킬, 카드 슬롯 크기와 이미지 표시 방식을 조정해 아이콘/카드가 잘리지 않도록 개선
- 모바일 화면에서 슬롯 액션 버튼이 숨지 않도록 가시성 보강

## 실행

```bash
npm ci --no-audit --no-fund
npm run build
```
