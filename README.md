# 소울 온라인 Alpha 0.37 Combat Flow Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.37은 0.36 UI 정리본을 기반으로, UI를 크게 흔들지 않고 전투 체감과 사냥 반복 플레이를 강화한 개발 패치입니다.

## 0.37 변경점

- 필드 전투 연쇄 시스템 추가
  - 제한 시간 안에 몬스터를 연속 처치하면 `CHAIN BONUS`가 유지됩니다.
  - 연쇄 수에 따라 EXP/GOLD 추가 보너스가 적용됩니다.
- 정예 몬스터 시스템 추가
  - `분노한`, `고대의`, `신속한`, `저주받은` 정예 개체가 확률적으로 등장합니다.
  - 보스/드래곤 계열은 정예 개체로 더 강하게 등장합니다.
  - 정예 개체는 이름, 오라, 타겟 패널 강조, 보상 배율이 적용됩니다.
- 필드 HUD에 체인 보너스 카드 추가
  - 기존 UI와 겹치지 않도록 우측 작은 카드 형태로 배치했습니다.
- 전투 로그 개선
  - 처치 로그에 연쇄 보너스와 최종 EXP/GOLD 보상이 표시됩니다.
- 정예 처치 시 전리품 팝업으로 추가 보상 체감 강화
- 버전, PWA 캐시, 에셋팩 메타데이터를 0.37 기준으로 정리

## 개발 메모

- 저장 데이터 구조는 변경하지 않았습니다.
- 저장 버전은 `SAVE_VERSION = 27` 유지입니다.
- 이번 패치는 콘텐츠 추가보다 전투 반복 플레이 감각을 강화하는 패스입니다.

## 실행

```bash
npm ci
npm run build
npm run dev
```

## 버전

- 앱 버전: `0.37.0`
- 저장 버전: `SAVE_VERSION = 27`
