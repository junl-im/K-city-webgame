# 소울 온라인 Alpha 0.48 Content Heaven + Anime Title + RPG Town Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.48은 사냥터/퀘스트 콘텐츠 총량을 크게 늘리고, 첫 시작 화면과 마을 화면의 비주얼 첫인상을 강화하는 대형 개발 패치입니다.

## 0.48 변경점

- 콘텐츠 천국 확장
  - 사냥터 13개 → 28개
  - 메인 스토리 46개 → 93개
  - 일일/순환 의뢰 30개 → 77개
- 신규 후반 사냥터 추가
  - 푸른 버섯 계곡
  - 잊힌 왕국 수로
  - 황혼 사막 관문
  - 얼어붙은 백야령
  - 별무덤 언덕
  - 칠흑 수도원
  - 천둥 고원
  - 고대왕의 회랑
  - 용암 제련소
  - 심연 해안
  - 몽환의 유적
  - 루미나 성역 심층
  - 일식 요새
  - 용척추 봉우리
  - 최후의 소울 전선
- 메인 퀘스트 후반부 대폭 추가
  - Lv.34 이후 장기 성장 구간
  - 보스 누적 토벌 목표
  - 드레이크/심연룡/야전 군주 반복 목표
  - 장비 공명, 스킬 숙련, 영혼 수집과 이어지는 장기 목표
- 시작 화면 비주얼 강화
  - 신규 오리지널 애니풍 캐릭터 SVG 일러스트 추가
  - 타이틀 광원, 소울 링, 버튼 광택 강화
- 마을 화면 비주얼 강화
  - 그림 속 RPG 마을 분위기로 재정리
  - 여관, 감시탑, 민가, 시장 가판대, 등불, 강, 다리 오브젝트 추가
  - 마을 사냥터 목록을 28개 전선에 맞게 동적 렌더링
- 앱 버전: `0.48.0`
- PWA 캐시: `soul-online-alpha-v0-48`
- 저장 버전: 구조 변경 없음, `SAVE_VERSION = 32` 유지
