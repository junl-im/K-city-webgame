# 소울 온라인 Alpha 0.33 Field Guide + Infernus Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.33은 0.32 그래픽 UI 스킨 위에 사냥터 추천 흐름과 고레벨 필드 비주얼을 보강한 개발 패치입니다.

## 0.33 변경점

- 사냥터 화면 상단에 현재 스토리 퀘스트 기반 `AUTO GUIDE` 추천 사냥터 카드 추가
- 완료 대기 중인 일일/순환 퀘스트 안내 문구 추가
- 사용자가 제공한 PVGames Infernus 무료 에셋 중 필드 장식용 리소스 선별 적용
- 용의 둥지, 악마의 균열, 잿불 능선, 혈석 광산, 수정 토벌대 등 고위험 사냥터에 인페르누스 타일/제단/화로/기둥/해골/암석 오브젝트 배치
- 인페르누스 사냥터에 붉은 열기/보라 균열 분위기 레이어 추가
- 사냥터 타일 선택 로직을 지역 테마 기반으로 분리하여 이후 설원/사막/던전 타일 확장 준비
- 버전, 저장 버전, PWA 캐시, 에셋팩 메타데이터를 0.33 기준으로 정리

## 0.32 기반 유지 내용

- Wenrexa UI 에셋 기반 그래픽 UI 스킨
- 필드 HUD, HP/MP/EXP 바, 몬스터 타겟창 그래픽 프레임
- 인벤토리/카드/스킬/영혼 슬롯 이미지 프레임
- 미니맵 비활성 유지
- 과한 중앙 확인 팝업 제거 유지
