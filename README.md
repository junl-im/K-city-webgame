# 소울 온라인 Alpha 0.42 Potion Belt + Field Menu Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.42는 0.41 Kakao HUD + Potion Pass 위에 자동 물약 벨트, 상급 물약, 상점 묶음 구매, 필드 메뉴 ID 충돌 정리를 추가한 개발 패치입니다.

## 0.42 변경점

- 필드 메뉴 버튼 ID 중복 제거
  - 상단 메뉴 버튼과 우측 하단 메뉴 버튼이 같은 `openMenu` ID를 쓰던 문제를 정리
  - 두 버튼 모두 `data-open-field-menu` 기반으로 안정적으로 동작
- 자동 물약 벨트 개선
  - HP/MP 단축 버튼이 하급 물약만 고정 사용하지 않고 현재 HP/MP 비율과 보유 수량에 맞춰 적절한 등급을 자동 선택
  - 자동사냥 중에도 하급/중급/상급 물약을 상황별로 선택
- 상급 생명/마나 물약 추가
  - 상급 생명 물약: HP 80% 회복
  - 상급 마나 물약: MP 85% 회복
- 물약 드랍 루프 보강
  - 초반 사냥터는 하급 물약
  - 중반 사냥터는 중급 물약
  - 고위험 사냥터는 상급 물약 드랍 가능
- 마을 상점 개선
  - 상급 물약 판매 추가
  - 소모품/재료 5개 묶음 구매 버튼 추가
- 인벤토리/상점 물약 벨트 요약 카드 추가
  - 보유 물약 수량
  - 등급별 회복량
  - 자동 선택 안내

## 빌드 확인

- 앱 버전: `0.42.0`
- 저장 버전: `SAVE_VERSION = 30`
- PWA 캐시: `soul-online-alpha-v0-42`
- `npm run build` 통과
