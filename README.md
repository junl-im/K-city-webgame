# 소울 온라인 Alpha 0.7 Cinematic Flow

모바일 웹 2.5D MMORPG 프로토타입입니다. 0.7은 첫 진입 흐름, 마을 허브 UI/UX, 필드 캐릭터·몬스터 표현, 전투 타격감, 몬스터 어그로 방식을 개선한 버전입니다.

## 0.7에서 바뀐 점

- 첫 화면을 **게임 시작** 버튼으로 시작하도록 변경
- 접속 흐름을 탭 이동 방식에서 순차 진행 방식으로 변경
  - 게임 시작 → 로그인 → 곰같은여우 서버 → 캐릭터 선택/생성 → 마을 입장
- 로그인 단계 표시를 버튼 탭이 아닌 진행 인디케이터로 변경
- 마을 화면 리디자인
  - 루미나 마을 디오라마 강화
  - 상단 캐릭터/재화 정보 정리
  - 사냥터 선택, 마을 기능, 빠른 액션 패널 재배치
  - 건물 라벨, NPC 느낌의 장식 요소 추가
- 필드 캐릭터/몬스터 SVG 아트 개선
  - 전사/술사/성직자 캐릭터 외형 보강
  - 슬라임/늑대/고블린/흑수정 곰/드래곤 외형 보강
- 전투 타격감 개선
  - 근접 돌진 모션 강화
  - 검격 이펙트 확대
  - 투사체 궤적과 성광 이펙트 보강
  - 피격 흔들림과 어그로 표시 개선
- 몬스터 동시 돌진 문제 완화
  - 사냥터별 몬스터 출현 제한
  - 초반 사냥터는 한 번에 1마리 위주로 반응
  - 레이드터만 최대 2마리까지 동시 교전
  - 몬스터별 감지 거리와 귀환 반경 적용
- 사냥터 입장 시 구역별 시작 좌표 적용

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 저장

- 로컬 저장은 브라우저 localStorage를 사용합니다.
- Firebase 설정이 있으면 `/users/{uid}`와 `/rankings/{uid}`에 클라우드 저장을 시도합니다.
- 기존 0.6 이하 캐릭터는 자동 마이그레이션됩니다.
