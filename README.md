# 소울 온라인 Alpha 0.57.0 Visual QA & Cinematic Polish Pass

0.57.0은 “보통의 웹 모바일 RPG처럼 첫 화면부터 전투까지 더 정돈되어 보이는가”를 기준으로 한 시각/UX 다듬기 패스입니다. 새 시스템을 무리하게 늘리지 않고, 타이틀·마을 NPC·필드 바닥 세트 드레싱·전투 타격 이펙트·슬롯 가독성을 집중 점검했습니다.

## 0.57.0 변경점

- 시작 화면을 0.57 시네마틱 히어로/루미나 파노라마로 갱신했습니다.
- 마을 NPC 브람/미온/세이라가 클릭 가능한 상호작용 포인트로 보이도록 마커와 연결 메뉴를 추가했습니다.
- 마을 무드 보드가 0.57 기준으로 갱신되어 타이틀, 마을, 전투, 필드 품질 체크를 한눈에 보여줍니다.
- 필드 오브젝트 6종을 추가하고 저레벨/중레벨/보스/고레벨 전선별 바닥 장식 밀도를 다시 조절했습니다.
- 타격 이펙트가 너무 난잡하게 겹치지 않도록 일반 타격은 가볍게, 크리티컬/스킬은 더 또렷하게 보이도록 임팩트 버스트를 재조정했습니다.
- 슬롯/상세창/터치 피드백의 이미지 영역과 글자 계층을 다시 정리했습니다.
- 긴 목록과 모바일 뷰에서 기존 성능 최적화 규칙을 유지했습니다.
