# 소울 온라인 Alpha 0.60.0 Visual Reboot Pass

0.60.0은 사용자가 제공한 고퀄리티 키비주얼을 실제 게임 시작 화면에 적용하고, 기존 SVG 장식형 타이틀/마을 표현을 걷어내는 비주얼 리부트 패치입니다.

## 핵심 변경점

- 업로드된 애니풍 키비주얼을 `title-keyvisual-060.webp`로 최적화해 첫 시작 화면 전체 배경으로 적용
- 기존 SVG 캐릭터 장식 타이틀을 숨기고, 실제 키비주얼 중심의 모바일 게임 타이틀 화면으로 재구성
- `TOUCH TO START` 버튼을 하단 글로우형 판타지 버튼으로 재디자인
- 같은 키비주얼 톤에서 마을 풍경용 `town-keyvisual-060.webp` 파노라마를 추출/보정
- 마을 화면을 단순 패널 배경이 아니라 그림 속 마을 풍경 위의 인터랙티브 지점 구조로 재구성
- 마을 주요 지점 핫스팟 추가: 필드 게이트, 대장간, 시장 상점, 소울 성소, 혈맹 성채
- 마을 UI 패널을 파란 하늘/화이트 글로우/판타지 유리 패널 톤으로 재정리
- 기존 과도한 SVG형 가짜 마을 장식은 0.60 화면에서는 숨김 처리
- 앱 버전 0.60.0, 저장 버전 44, PWA 캐시 soul-online-alpha-v0-60

## 검증

- npm ci
- npx tsc --noEmit
- npm run build
