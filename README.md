# 소울 온라인 Alpha 0.41 Kakao HUD + Potion Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.41은 0.40 Gear Resonance Pass 위에 카카오톡 인앱 브라우저 대응, 필드 HUD 재배치, 접속 플로우 단축, 캐릭터 프로필 이미지 표시, 물약 시스템을 추가한 개발 패치입니다.

## 0.41 변경점

- 카카오톡 인앱 브라우저 감지 및 안전 HUD 배치 추가
- 전체화면/세로 방향 잠금 실패 시에도 화면을 막지 않고 플레이 가능하도록 우회
- 캐릭터 선택 및 캐릭터 생성 후 바로 마을 접속
- 필드 캐릭터 정보창을 상단 전체 폭으로 확장
- 몬스터 타겟창을 작게 줄이고 캐릭터 정보창 아래 중앙 배치
- 퀘스트 진행창을 우측 상단으로 이동
- 우측 상단 메뉴 버튼을 우측 하단 액션 묶음으로 이동
- 우측 하단 액션 버튼 정리: 자동 / 공격 / 메뉴, 위쪽 가방, 그 위 절전
- 스킬 버튼을 액션 버튼 위쪽으로 재배치
- HP/MP 물약 단축 버튼 추가
- 하급/중급 HP/MP 물약 아이템 및 상점 판매 추가
- 자동사냥 중 HP/MP가 낮으면 하급 물약 자동 사용
- 캐릭터 HUD, 마을 상단 카드, 계정 프로필에 실제 직업/성별 초상화 표시

## 실행

```bash
npm ci
npm run build
npm run dev
```
