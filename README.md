# 소울 온라인 Alpha 0.58.0 Immersive Refinement Pass

0.58.0은 0.57의 시각 QA 위에 이어지는 몰입형 다듬기 패스입니다. 새 기능을 무리하게 벌리기보다, 모바일 RPG에서 중요한 첫인상, 마을 상호작용, 전투 타격 가독성, 보스 드랍 연출, 필드 바닥/오브젝트 품질을 다시 정리했습니다.

## 0.58.0 변경점

- 마을 NPC 상호작용을 짧은 시네마틱 대화 카드로 보강
- 타이틀 히어로/마을 파노라마 0.58 신규 SVG 추가
- 신규 필드 소품 6종과 전장 조명 워시 추가
- 보스/후반 전선용 장식 밀도와 바닥 오브젝트 재배치
- 전투 타격 이펙트 과밀도 완화와 강타/희귀 드랍 강조 유지
- 신규 사냥터 4개, 메인 퀘스트 16개, 의뢰 5개 추가
- 신규 장비/재료/상자 아이템 8종과 아이콘 추가
- 가방/상세창 아이템 아트 버전 갱신

## 검증

- `npm ci`
- `npx tsc --noEmit`
- `npm run build`
