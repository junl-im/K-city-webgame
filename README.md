# 소울 온라인 Alpha 0.8 Content Foundation

모바일 웹 2.5D MMORPG 프로토타입입니다. 0.8은 화면 흐름과 마을 UX를 더 세련되게 다듬고, 캐릭터별로 저장되는 **메인 스토리 퀘스트 시스템**을 추가한 버전입니다.

## 0.8에서 바뀐 점

- 첫 화면 연출 강화
  - `게임 시작` 중심의 타이틀 화면 유지
  - Alpha 0.8 표기, 미니 메뉴, 세계관 문구 개선
- 로그인/서버/캐릭터 UI 개선
  - 로그인 화면을 세계관 카드 + 접속 플로우 카드로 분리
  - 탭처럼 보이던 진행 단계를 순차 접속 인디케이터로 정리
  - 직업명을 소울 온라인 분위기에 맞게 변경
    - 전사 → 검혼
    - 술사 → 영술사
    - 성직자 → 성휘사
- 루미나 마을 화면 보강
  - 마을 메인 패널에 현재 스토리 스냅샷 추가
  - 스토리 진행 버튼 추가
  - 사냥터 잠금/해금 상태 표시
  - 마을 기능에 `스토리` 메뉴 추가
  - 전체적인 카드, 배경, 글래스 패널 스타일 개선
- 메인 스토리 퀘스트 시스템 추가
  - 캐릭터별 `story` 진행도 저장
  - 대화형 퀘스트, 처치 퀘스트, 레벨 달성 퀘스트 지원
  - 스토리 보상: EXP, 골드, 소울젬, 아이템
  - 진행 중 퀘스트의 목표/진행률/보상 표시
- 신규 스토리 챕터
  - CH.1 꺼져가는 루미나의 등불
  - CH.1 숲 입구의 첫 정화
  - CH.2 수정 늑대의 울음
  - CH.3 고블린 길목의 검은 깃발
  - CH.4 소울 코어 공명
  - CH.5 흑수정 곰의 심장
- 저장 버전 갱신
  - `SAVE_VERSION = 5`
  - 기존 0.7 이하 캐릭터는 자동 마이그레이션됩니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 저장

- 로컬 저장은 브라우저 localStorage를 사용합니다.
- Firebase 설정이 있으면 `/users/{uid}`와 `/rankings/{uid}`에 클라우드 저장을 시도합니다.
- 클라우드 저장 실패 시 로컬 저장은 유지되고, 자동 저장 실패 메시지는 반복 노출되지 않습니다.
