# 소울 온라인 Alpha 0.23 Real Visual Asset Expansion Pass

모바일 세로형 2.5D 웹 RPG 프로토타입입니다. 0.23에서는 타이틀/필드 HUD 복구, 진행중 퀘스트 HUD, 조이스틱/전투 버튼 재배치, 고난도 사냥터 확장, 그리고 대규모 오리지널 2.5D 에셋팩을 적용했습니다.

## 0.23 변경점

- 게임 시작/로그인 전 HUD 노출 문제 수정
- 마을 하단 5버튼 메뉴는 마을에서만 표시
- 사냥터에서는 전투 HUD, 조이스틱, 자동/공격/가방/카드/절전 버튼만 표시
- EXP 바를 최상단 얇은 바 형태로 재배치
- 사냥터 진행중 퀘스트 HUD 추가
- 좌측 프로필 클릭 시 계정/캐릭터 정보창 열림
- 조이스틱 위치와 터치 영역 재보정
- 캐릭터 이동속도 하향 조정
- 스토리 완료 후 다음 진행 사냥터로 자동 워프 보강
- 고레벨 사냥터 3종 추가
  - 혈석 광산 심층
  - 천공 성채 외곽
  - 마왕의 균열
- 8방향 캐릭터/몬스터 시트 해상도 상향
- 몬스터별 전용 다방향 시트 추가
- 나무 10종, 바위 10종, 상자 5종, 횃불 5종, 마을 건물 4종, UI 아이콘 60종, 카드/영혼/무기/스킬 이펙트 리소스팩 추가

## 에셋 안내

이번 에셋은 미르/뮤/리니지/오딘 계열의 어두운 2.5D 판타지 분위기를 참고한 **오리지널 생성형 샘플 리소스**입니다. 실제 상용 원화 복제물이 아니며, 같은 파일명으로 외주/AI 프리렌더 원본을 교체하면 바로 런타임에 적용됩니다.
