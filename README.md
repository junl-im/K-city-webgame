# 소울 온라인 Alpha 0.43 Lineage HUD + Potion Balance Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.43은 0.42 Potion Belt 패치 이후 남은 사냥터 인터페이스 과밀, 슬롯 가독성, 상세 정보 접근성, 물약 밸런스를 정리한 개발 패치입니다.

## 0.43 변경점

- 사냥터 HUD를 리니지식 모바일 MMORPG 구조에 맞춰 재배치
  - 캐릭터 정보창을 상단 전체 폭의 얇은 바 형태로 정리
  - 상단 햄버거 메뉴 버튼 숨김
  - 몬스터 타겟 정보창을 축소하고 캐릭터 정보창 아래 중앙 배치
  - 퀘스트 진행창을 우측 상단 작은 카드로 이동
  - 콤보 패널을 캐릭터 초상화 우측 아래의 작은 반투명 pill 형태로 이동
- 우측 하단 조작부 정리
  - 반투명 메뉴 배경 제거
  - 자동 / 공격 / 더보기 / 가방 / 절전을 개별 원형 버튼으로 정리
  - 스킬 버튼을 우측 하단 액션 버튼 위쪽으로 정렬
- 슬롯 UX 재정리
  - 가방, 카드, 스킬 슬롯 아이콘 면적 확대
  - 슬롯 내부 직접 버튼 제거
  - 한 번 클릭하면 상세창이 열리고, 상세창에서 장착/해제/강화/사용 처리
  - 모바일에서 hover 버튼이 안 보여 기능을 못 찾는 문제 완화
- 상세 정보창 보강
  - 카드/장비/소모품/스킬 상세 액션을 모달 안으로 집중
  - 스킬 상세창에서 숙련 강화 가능
- 물약 밸런스 조정
  - 하급 물약: 2% 회복
  - 중급 물약: 3.5% 회복
  - 상급 물약: 5% 회복
  - 자동 물약 판단 간격과 짧은 재사용 대기시간 추가
  - 회복량이 낮아진 만큼 상점 가격 하향
  - 신규 캐릭터 기본 물약 수량 상향
