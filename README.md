# Soul Online Alpha 0.82 덮어쓰기 패치

이 패치는 0.81 판타지 UI/에셋화 패치 위에 그대로 복사해서 덮어쓰는 **소울 온라인 UI 다듬기 패치**입니다.

`K-city`는 이전 업로드 ZIP/작업 폴더명에서 이어진 임시 작업명일 뿐이며, 게임명과 사용자 노출 브랜드는 **Soul Online / 소울 온라인**입니다.

## 적용 순서

1. GitHub Desktop에서 현재 상태를 먼저 커밋합니다.
2. 이 ZIP의 내용을 프로젝트 루트에 그대로 복사하고 같은 파일은 덮어씁니다.
3. 터미널에서 `npm install` 후 `npm run build`를 실행합니다.
4. Firebase Hosting을 쓰는 경우 `firebase deploy --only hosting`으로 배포합니다.

## 이번 패치 핵심

- 프로젝트 버전을 `0.82.0`으로 갱신했습니다.
- PWA 캐시를 `soul-online-alpha-v0-82`로 갱신했습니다.
- `fantasy-ui-082`, `title-screen-082`, `login-screen-082`, `town-screen-082` 스킨 레이어를 추가했습니다.
- 마을 드로어 상단에 `town-flow-advisor-082` 진행 가이드를 추가했습니다.
- 현재 스토리 목표, 진행도, 추천 사냥터, 전투력, 보상 대기, 가방 사용량을 한눈에 보이게 했습니다.
- 마을 로비 버튼, 재화 바, 하단 메뉴, 상점/가방/스토리 행을 판타지 블루/골드 톤으로 더 정리했습니다.
- 모바일 safe-frame을 강화해 UI가 화면 밖으로 밀리는 문제를 더 줄였습니다.
- 필드 HUD, 미니맵, 전투 로그, 조이스틱, 스킬/공격 독의 충돌을 한 번 더 완화했습니다.

## 다음 우선순위

0.83에서는 가방, 스킬, 상점, 보상 팝업을 같은 UI 키트로 완전히 통일하는 것이 좋습니다. 동시에 `styles.css`와 `main.ts`가 너무 커진 상태이므로, 화면 단위 분리 작업을 시작해야 합니다.
