# 소울 온라인 Alpha 0.13 Field Quality Pass

모바일 세로형 2.5D 웹 MMORPG 프로토타입입니다. 0.13에서는 이전 0.11~0.12의 BGM/효과음/비주얼 패스 위에 **필드 바닥 퀄리티**, **지역별 지형감**, **절벽/물가 경계**, **흙길/수정 이끼/흑수정 지형**, **맵 장식 프랍**을 추가했습니다.

## 실행

```bash
npm ci
npm run dev
```

## 빌드

```bash
npm run build
```

## 0.13 Field Quality Pass

- 타일 종류 확장
  - 잔디
  - 흙길
  - 수정 이끼
  - 석재/폐허 바닥
  - 흑수정 지형
  - 물가
  - 절벽
  - 포탈
- 반복 타일 느낌 완화
  - 타일 위 랜덤성 있는 풀잎/자갈/균열/광맥 데칼 추가
  - 물가와 절벽 주변 경계 하이라이트 추가
  - 흙길과 잔디 경계가 더 자연스럽게 보이도록 보조 라인 추가
- 맵 배치 개선
  - 루미나 마을 중심부는 석재 광장
  - 초반 사냥터는 잔디와 흙길 중심
  - 수정 이끼길은 푸른 이끼/광맥 중심
  - 고블린 길목과 흑수정 동굴은 폐허/돌바닥 중심
  - 수정 레이드 터는 흑수정 제단 느낌 강화
- 신규 프랍
  - 바위
  - 폐허 기둥
- 지역 분위기 강화
  - 레이드터 제단 링
  - 흑수정 동굴 광원
  - 지형 백드롭 그림자

## 이전 주요 기능 유지

- 첫 화면 → 로그인 → 서버 → 캐릭터 → 마을 입장 플로우
- 루미나 마을 허브
- 스토리 퀘스트
- 사냥터 지역 진행도
- 몬스터 AI 상태머신
- 스킬 버튼/쿨타임/MP 소모/범위 공격
- 카드 세트 효과
- 장비 강화
- BGM/SFX 및 사운드 설정
- PWA 세로 모드 설정
