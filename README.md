# 소울 온라인 Alpha 0.21 Combat / Content Expansion Pass

모바일 세로형 2.5D 웹 RPG 프로토타입입니다. 0.21에서는 몬스터 난이도, 스킬 습득 구조, 강화 20강, 카드/영혼/장비 다양화, 필드보스, 절전 모드, 스킬 이펙트, 닉네임 필터, UI 정리를 집중 적용했습니다.

## 0.21 변경점

- 몬스터 난이도 상승: 강화와 카드 세팅이 필요하도록 HP/ATK/DEF 재조정
- 강화 시스템 확장: +20까지 가능, +4까지 100%, 이후 단계별 10%씩 하락, 최저 5%
- +10 이후 강화석 재료 사용
- 초기 스킬 미습득 상태로 시작
- 첫 스토리 보상 또는 상점 스킬서/드랍 스킬서로 스킬 습득
- 자동사냥 중 배운 스킬 자동 사용 유지
- 스킬별 이펙트 차별화: 천뢰검, 낙화염구, 천뇌폭우, 성광 계열
- 카드 다양화 및 4x9 카드 슬롯 UI
- 인벤토리 7x7 슬롯 UI
- 영혼 종류 확장
- 몬스터 12종 체계로 확장
- 사냥터 10종 구성
- 필드보스 야전 군주 발타르 추가
- HP/MP는 캐릭터 정보와 HUD에 표시, 경험치는 길고 얇은 게이지로 유지
- 절전 모드 오버레이 추가
- 닉네임 생성 금지어 필터 추가
- 타이틀/마을 소개 문구 정리
- BGM 루프를 더 부드러운 MR형 OGG로 재생성
