# 소울 온라인 Alpha 0.62.0 Reference Visual Update Pass

0.62.0은 첨부 레퍼런스처럼 밝은 하늘색/아이보리/금장 프레임을 중심으로 UI, 타이틀, 마을, 필드 전투 표현을 한 단계 올리는 패치입니다.

핵심 방향은 명확합니다. 저급 임시 이미지를 덧씌우지 않고, 기존 고해상도 WebP 스프라이트/키비주얼을 기반으로 화면 구성, 프레임, 조명, 캐릭터/몬스터 실루엣, 필드 무드 레이어를 강화합니다.

## 주요 변경

- 타이틀 화면에 0.62 레퍼런스급 밝은 판타지 UI 스킨 적용
- 로그인/서버/캐릭터 선택 화면을 아이보리 패널 + 블루 버튼 + 금장 프레임 방향으로 재정리
- 마을 화면에 0.62 상단 카드, 화폐 칩, 핫스팟, 기능 패널 스킨 추가
- 필드 HUD, 조이스틱, 전투 버튼, 시트/슬롯/상세 카드에 동일한 프리미엄 스킨 적용
- Pixi 필드에 0.62 레퍼런스 조명 패스 추가: 하늘 반사, 보스/스폰 포켓 광원, 경로 리본
- 플레이어/몬스터 스프라이트 표시를 확대하고 바닥 원형 프레임, 네임플레이트, 엘리트/보스 베이스 플레어 강화
- 앱 버전 0.62.0, PWA 캐시 soul-online-alpha-v0-62

## 검증

- npm ci
- npm run build
