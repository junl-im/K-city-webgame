# 소울 온라인 Alpha 0.32 Graphic UI Skin Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.32는 0.31의 사냥터 HUD 정리, 미니맵 제거, 퀘스트/슬롯 개선 위에 사용자가 제공한 무료 UI 에셋팩을 실제 런타임 UI 스킨으로 적용한 그래픽 UI 패스입니다.

## 0.32 변경점

- Wenrexa 계열 UI 에셋을 선별해 `src/assets/ui/wenrexa`에 정리
- HUD, 몬스터 타겟 카드, 재화 표시, 퀘스트 트래커를 이미지 프레임 기반으로 재스킨
- 인벤토리, 카드, 스킬, 영혼 슬롯에 그래픽 슬롯 프레임 적용
- HP/MP/EXP/퀘스트 진행바를 이미지 테두리형 프로그레스 바로 변경
- 전투 스킬 단축창에 실제 스킬 아이콘 이미지를 표시하도록 개선
- 마을 패널, 하단 메뉴, 드로어, 상세 팝업, 토스트, 전리품 팝업까지 동일한 판타지 UI 톤으로 통일
- 닫기 버튼은 이미지 아이콘으로 변경
- 미니맵은 0.31 방침대로 계속 비활성 유지

## 0.31 기반 유지 내용

- 사냥터에서 플레이어 HP/MP 카드와 몬스터 타겟 카드가 겹치지 않도록 HUD 배치 재정렬
- 위치가 불안정하고 시야를 방해하던 미니맵 UI 제거
- 퀘스트 스토리 46개, 일일/순환 퀘스트 30개로 대폭 확장
- 필드 바닥 오브젝트를 지역별 큐레이션 방식으로 재배치
- 버튼 클릭마다 뜨던 중앙 확인 피드백 팝업 제거
- 인벤토리, 스킬, 카드 슬롯 크기와 이미지 표시 방식 개선
