# 소울 온라인 Alpha 0.38 Monster Pattern Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.38은 0.37 Combat Flow 위에 몬스터 경고 패턴과 보스전 회피 타이밍을 추가한 개발 패치입니다.

## 0.38 변경점

- 고위험 몬스터 패턴 시스템 추가
  - 화염 드레이크 / 심연룡: 직선형 화염 숨결
  - 야전 군주 / 흑요석 가디언 / 묘지 기사: 근접 충격파
  - 망령 사제 / 폭풍 하피: 지정 지점 그림자 폭발
- 패턴 시전 전 바닥/라인 경고 텔레그래프 표시
- 경고 범위에서 벗어나면 `회피` 피드백 표시
- 패턴 적중 시 일반 공격보다 강한 피해, 화면 흔들림, 이펙트 출력
- 정예 몬스터와 보스는 HP가 낮아지면 패턴 재사용 간격이 약간 짧아짐
- 몬스터 패턴 상태를 기존 AI와 분리해 이동/일반 공격과 충돌하지 않도록 정리
- 사냥터 UI는 0.36~0.37 안정화 규격을 유지하고 추가 창/버튼은 만들지 않음
- 버전, PWA 캐시, 에셋팩 메타데이터를 0.38 기준으로 정리

## 조작

- 터치/마우스: 필드 이동
- 공격 버튼: 기본 공격
- 스킬 버튼 1~3: 직업 스킬
- 자동사냥: 타겟 추적 및 스킬 자동 사용
