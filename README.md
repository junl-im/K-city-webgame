# 소울 온라인 Alpha 0.61.0 Raster Visual Pipeline Pass

0.61.0은 0.60 비주얼 리부트 위에 이어지는 래스터 기반 비주얼 파이프라인 패치입니다.

핵심 방향은 명확합니다. 활성 타이틀/마을/UI 비주얼은 SVG 장식이 아니라 WebP/PNG/JPG 기반 키비주얼과 텍스처로 구성합니다.

## 주요 변경

- 타이틀 화면 0.61 WebP 키비주얼 적용
- 가로 화면용 타이틀 WebP 배경 추가
- 타이틀 블러 백플레이트 WebP 추가
- 마을 배경을 WebP 키비주얼 기반으로 재구성
- 마을 주요 지점 핫스팟을 래스터 핀/패널 텍스처로 교체
- 패널/버튼/슬롯 기본 텍스처 WebP 3종 추가
- 시작 버튼, 마을 패널, 가방/상세 슬롯 톤을 키비주얼과 맞춤
- 기존 SVG형 타이틀/마을 장식은 0.61 활성 화면에서 숨김 처리
- 앱 버전 0.61.0, 저장 버전 45, PWA 캐시 soul-online-alpha-v0-61

## 검증

- npm ci
- npx tsc --noEmit
- npm run build
