# 소울 온라인 Alpha 0.44 Opening Illustration + Silent In-App Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.44는 0.43 Lineage HUD 패치 이후 첫 시작 화면의 인상을 강화하고, 카카오톡/인앱 브라우저 대응 문구가 사용자에게 노출되지 않도록 정리한 개발 패치입니다.

## 0.44 변경점

- 첫 시작 화면에 신규 캐릭터 일러스트 추가
- 타이틀 화면 배경, 룬, 영혼 링, 시작 버튼 연출 보강
- 카카오톡/인앱 브라우저 대응 로직은 유지
- 대응 중/안전 배치/전체화면 제한 같은 개발자용 안내 배너 및 토스트 제거
- 시작 화면의 브라우저 정책 안내 문구 제거
- 가로 뷰포트에서도 시작 화면 구성이 크게 무너지지 않도록 보정

## 빌드

```bash
npm ci
npm run build
```
