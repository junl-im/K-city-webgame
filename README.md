# 소울 온라인 Alpha 0.9 Content Expansion

모바일 웹 2.5D MMORPG 프로토타입입니다. 0.9는 0.8의 스토리/마을 기반 위에 **사냥터 지역 진행도**, **몬스터 AI 상태머신**, **전투 타격감 강화**, **슬롯형 카드/인벤토리/스킬 UI**, **자동사냥 경로 복구**를 추가한 버전입니다.

## 0.9에서 바뀐 점

- 사냥터 지역 진행도 확장
  - 초록 숲 입구
  - 수정 이끼길
  - 고블린 길목
  - 흑수정 동굴
  - 수정 레이드 터
  - 각 지역은 권장 레벨, 해금 조건, 몬스터 구성을 갖습니다.
- 몬스터 AI 상태머신 추가
  - `idle` 배회
  - `alert` 발견/경고 딜레이
  - `chase` 추격
  - `attackWindup` 공격 준비 모션
  - `attack` 타격
  - `return` 원위치 복귀
- 몬스터 몰림 완화
  - 일반 사냥터는 1마리 위주로 교전
  - 흑수정 동굴/레이드 터는 최대 2마리까지 교전
- 전투 타격감 강화
  - 공격 준비 이펙트
  - 피격 히트스톱
  - 공격 경고/어그로 링 개선
  - 몬스터 복귀 상태 표시
- 슬롯형 UI 확장
  - 카드 도감: 3x3 슬롯
  - 스킬: 3x3 슬롯
  - 인벤토리: 5x9 슬롯
  - 마을 패널과 필드 메뉴 모두 슬롯형 표시를 사용합니다.
- 자동사냥 멈춤 방지
  - 직선 이동이 막히면 측면 우회 좌표 탐색
  - 일정 시간 제자리면 타겟/경로 재계산
  - 맵 지형에 끼인 경우 `자동사냥 경로 재탐색` 로그 표시
- 저장 버전 갱신
  - `SAVE_VERSION = 6`
  - 기존 0.8 이하 캐릭터는 자동 마이그레이션됩니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 저장

- 로컬 저장은 브라우저 localStorage를 사용합니다.
- Firebase 설정이 있으면 `/users/{uid}`와 `/rankings/{uid}`에 클라우드 저장을 시도합니다.
- 클라우드 저장 실패 시 로컬 저장은 유지되고, 자동 저장 실패 메시지는 반복 노출되지 않습니다.
