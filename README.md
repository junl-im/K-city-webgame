# 소울 온라인 Alpha 0.47 Motion + Field Object Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.47은 0.46 자동사냥/보스 보상 루프 이후, 사냥터 체감과 필드 비주얼을 크게 끌어올리는 비주얼 개발 패치입니다.

## 0.47 변경점

- 전투 모션 피드백 강화
  - 기본 공격 시 캐릭터 잔상 추가
  - 근접 공격 slash 스프라이트 이펙트 추가
  - 원거리 투사체에 fireball/lightning 스프라이트 적용
  - 성직자 공격/회복 계열에 holy nova 스프라이트 적용
  - 스킬 폭발에 직업별 전용 이펙트 스프라이트 합성
  - 몬스터 처치 시 영혼 방출 연출 추가

- 이동/사냥 체감 개선
  - 이동/자동사냥 중 발밑 먼지 이펙트 추가
  - 자동사냥 달리기와 수동 이동 먼지 간격 분리
  - 공격 자세 시작 시 짧은 afterimage로 타격감을 보강

- 필드 오브젝트 확장
  - 신규 장식 에셋 추가: 덤불, 버섯, 깃발, 뼈더미, 수정광맥, 룬석, 그루터기
  - 지역별 장식 배치 패스 추가
  - 초반 숲, 수정지대, 폐허길, 검은 동굴, 지옥/균열 지역의 오브젝트 분위기 분리
  - 지역별 빛 입자/마력 안개 레이어 추가

- 안정성
  - UI 레이아웃은 0.46 기준 유지
  - 저장 구조 변경 없음
  - 저장 버전 `SAVE_VERSION = 32` 유지
  - 앱 버전: `0.47.0`
  - PWA 캐시: `soul-online-alpha-v0-47`

## 실행

```bash
npm ci
npm run build
npm run dev
```
