# 소울 온라인 Alpha 0.39 Skill Mastery Pass

모바일 2.5D 웹 MMORPG 프로토타입입니다. 0.39는 0.38 Monster Pattern Pass 위에 스킬 숙련 성장과 자동사냥 스킬 판단 개선을 추가한 개발 패치입니다.

## 0.39 변경점

- 스킬 숙련 레벨 시스템 추가
  - 습득한 스킬은 Lv.1부터 Lv.5까지 강화 가능
  - 숙련 레벨마다 피해/회복량 증가, 쿨타임 감소, MP 소모 감소
- 스킬창 UI 정리
  - 스킬 카드에 현재 숙련 레벨, 다음 비용, 효율 변화 표시
  - 마을 스킬창과 필드 스킬 시트 양쪽에서 숙련 강화 가능
- 스킬 상세 팝업 보강
  - 숙련 효과, 현재 MP/쿨타임, 다음 강화 비용 표시
- 필드 전투 스킬 로직 보강
  - 실제 전투 피해/회복, 쿨타임, MP 소모에 숙련 효과 반영
  - 자동사냥이 강화된 실제 MP 소모 기준으로 스킬 사용 판단
- 저장 데이터 마이그레이션
  - `skillLevels` 저장 구조 추가
  - 기존 저장은 자동으로 스킬 Lv.1 기준으로 보정
- 버전, PWA 캐시, 에셋팩 메타데이터를 0.39 기준으로 정리

## 실행

```bash
npm ci
npm run build
npm run dev
```
