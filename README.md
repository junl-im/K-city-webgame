# 소울 온라인 Alpha 0.10 Growth & Skills

모바일 세로형 2.5D 웹 MMORPG 프로토타입입니다. 0.10은 0.9의 지역 진행도/AI/슬롯 UI 위에 **카드 세트 효과**, **장비 강화**, **실제 액티브 스킬**, **세로모드 고정 보강**, **마을 패널 크기 안정화**를 추가한 버전입니다.

## 0.10에서 바뀐 점

- 카드 세트 효과 추가
  - 초록 숲의 기억
  - 소울 바인더의 서약
  - 흑수정의 분노
  - 장착 카드 조합이 맞으면 능력치 보너스가 자동 발동합니다.
- 장비 강화 시스템 추가
  - 무기/방어구/유물을 +10까지 강화할 수 있습니다.
  - 강화는 골드와 소울 파편을 사용합니다.
  - 강화 보너스는 전투력, HP, ATK, DEF 계산에 반영됩니다.
- 스킬 실제 전투 연결
  - 전투 화면 우측에 1/2/3 스킬 버튼 추가
  - 스킬별 쿨타임 적용
  - MP 소모 적용
  - 범위 피해, 회복형 스킬, 회복+피해형 스킬 적용
- 마을 패널 크기 안정화
  - 스토리/퀘스트 보상 화면이 뷰포트를 밀어 화면이 커지는 현상을 줄이도록 마을 화면과 패널 높이를 `100dvh` 기준으로 고정했습니다.
  - 긴 보상/퀘스트 문구는 내부 스크롤과 줄바꿈으로 처리합니다.
- 모바일 세로 모드 보강
  - PWA manifest orientation을 `portrait`로 변경했습니다.
  - 전체화면 진입 시 Screen Orientation API로 세로 잠금을 시도합니다.
  - 가로로 돌아간 경우 세로 모드 안내 오버레이를 표시합니다.
- 저장 버전 갱신
  - `SAVE_VERSION = 7`
  - 기존 0.9 이하 캐릭터는 자동 마이그레이션됩니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 저장

- 로컬 저장은 브라우저 localStorage를 사용합니다.
- Firebase 설정이 있으면 `/users/{uid}`와 `/rankings/{uid}`에 클라우드 저장을 시도합니다.
- 클라우드 저장 실패 시 로컬 저장은 유지되고, 자동 저장 실패 메시지는 반복 노출되지 않습니다.
